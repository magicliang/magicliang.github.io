Throwing exception is expensive.

  иҝҷеҸҘиҜқеңЁ Java зҡ„зЁӢеәҸе‘ҳдё–з•ҢйҮҢйқўе·Із»ҸжҲҗдёәиҖҒз”ҹеёёи°ҲгҖӮеҚҙеҫҲе°‘жңүдәәи°ҲеҸҠпјҢдҪҶеҲ°еә•жҠӣеҮәејӮеёёжҜ”жӯЈеёёзҡ„зЁӢеәҸи·іиҪ¬иҝ”еӣһж…ўеңЁе“ӘйҮҢпјҢжңүеӨҡж…ўгҖӮвҖңдёҚиҰҒж»Ҙз”ЁејӮеёёвҖқеҘҪеғҸдёҖдёӘзҢҙеӯҗе®ҡеҫӢпјҢдәәд»¬еҝҳи®°дәҶдёәд»Җд№ҲдёҚиғҪиҝҷд№ҲеҒҡпјҢеҚҙдёҚжҳҺзҷҪдёәд»Җд№ҲдёҚиғҪиҝҷд№ҲеҒҡгҖӮ

  иҝҷеҮ еӨ©иҜ»дәҶдёҖдҪҚеҗҢдәӢеҶҷзҡ„еҘҪж–Ү[гҖҠJavaиҷҡжӢҹжңәжҳҜеҰӮдҪ•еӨ„зҗҶејӮеёёзҡ„гҖӢ][2]пјҢж·ұе…Ҙең°еҲҶжһҗдәҶ JVM еҜ№ејӮеёёи·іиҪ¬зҡ„еӨ„зҗҶиҝҮзЁӢ: JVM дјҡйҖҡиҝҮејӮеёёиЎЁзҡ„жңәеҲ¶пјҢдјҳеҢ–ејӮеёёжҠӣеҮәе’ҢжӯЈеёёиҝ”еӣһд№Ӣй—ҙзҡ„жҖ§иғҪе·®ејӮгҖӮд»…д»ҺзЁӢеәҸи®Ўж•°еҷЁзҡ„з§»еҠЁдёҠжқҘи®ІпјҢжҠӣеҮәдёҖдёӘејӮеёёеҜ№ж Ҳеё§зҡ„еј№ж Ҳе№¶дёҚжҜ”зӣҙжҺҘиҝ”еӣһжӣҙжҳӮиҙөгҖӮеҶҷеңЁеүҚеӨҙзҡ„з»“и®әжҳҜпјҡвҖңtry-catchиҜӯеҸҘеқ—еҮ д№ҺдёҚдјҡеҪұе“ҚзЁӢеәҸиҝҗиЎҢжҖ§иғҪпјҒеңЁејҖеҗҜJITзҡ„жғ…еҶөдёӢпјҢthrowд№ҹдёҚдјҡеўһеҠ еӨҡе°‘зі»з»ҹејҖй”ҖгҖӮвҖқе®һйҷ…дёҠиҝҷзҜҮж–Үз« д№ҹеҒҡдәҶдёҖдәӣеҜ№жҜ”пјҢеңЁдёҚеҗҢзҡ„еңәжҷҜдёӢпјҢtry-catch дјҡдёҚдјҡи®©зі»з»ҹеҸҳж…ўгҖӮ

AжҜҸж¬ЎйғҪжҠӣејӮеёё AиҖ—ж—¶зәҰжҳҜBзҡ„30еҖҚ дёӨиҖ…иҖ—ж—¶еҮ д№ҺзӣёеҗҢ

  иҝҷеҮ д№ҺжҺЁзҝ»дәҶжҲ‘д»¬ж—ўжңүзҡ„еҲ»жқҝеҚ°иұЎпјҢд»ҺжӯӨжҠӣеҮәејӮеёёдёҚеҶҚжҳҜдёҖдёӘйңҖиҰҒиҖғиҷ‘жҖ§иғҪзҡ„и®ҫи®ЎеҶіе®ҡдәҶгҖӮеңЁд»”з»Ҷз ”з©¶дәҶиҝҷдёӘй—®йўҳд»ҘеҗҺпјҢжҲ‘еҚҙжңүдәҶдёҖдёӘдёҚеҗҢзҡ„з»“и®әпјҡtry-catch иҜӯеҸҘеңЁ jit зҡ„её®еҠ©дёӢ пјҢд№ҹи®ёеҸҜд»ҘиҫҫеҲ°е’ҢжӯЈеёё return дёҖж ·зҡ„жҖ§иғҪ пјҢз„¶иҖҢ throw еҚҙдјҡдә§з”ҹиҝңжҜ”ж–ҮдёӯжҸҸиҝ°зҡ„жӣҙдёҘйҮҚзҡ„жҖ§иғҪеҪұе“Қ пјҢеӣ дёә throw дёҚжҳҜеӯӨз«Ӣзҡ„иҜӯеҸҘпјҢе®ғеҝ…йЎ»дјҙйҡҸзқҖејӮеёёеҜ№иұЎзҡ„еҲӣе»әпјҢиҖҢејӮеёёеҜ№иұЎзҡ„еҲӣе»әзҡ„жҳӮиҙөд»Јд»·пјҢжҳҜдёҚеҸҜиғҪиў« jit дјҳеҢ–жҺүзҡ„гҖӮд№ҹе°ұжҳҜиҜҙпјҢжҲ‘и®Өдёә[гҖҠJavaиҷҡжӢҹжңәжҳҜеҰӮдҪ•еӨ„зҗҶејӮеёёзҡ„гҖӢ][3]дёӯз»“и®әзҡ„еүҚеҚҠйғЁеҲҶжҳҜжӯЈзЎ®зҡ„пјҢеҗҺеҚҠйғЁеҲҶжҳҜдёҚеҮҶзЎ®зҡ„гҖӮ

ејӮеёёзҡ„жңәеҲ¶

  JVM зҡ„ејӮеёёеӨ„зҗҶжңәеҲ¶пјҢеӨ§иҮҙеҸҜд»ҘеҲҶдёәдёүдёӘйғЁеҲҶ пјҡ

  1. new Exception
  2. throw Exception
  3. catch and deal with Exception

  йҖҡиҝҮ[гҖҠJavaиҷҡжӢҹжңәжҳҜеҰӮдҪ•еӨ„зҗҶејӮеёёзҡ„гҖӢ][4]жҲ‘д»¬е·Із»ҸеҸҜд»ҘжҳҺзЎ®зҗҶи§ЈпјҢJVM еҜ№дәҺ try-throw-catch зҡ„зЁӢеәҸжҺ§еҲ¶жөҒеӨ„зҗҶпјҢдёҺжҷ®йҖҡзҡ„ return еҰӮеҮәдёҖиҫҷпјҢйғҪжҳҜеҹәдәҺзЁӢеәҸи®Ўж•°еҷЁзҡ„ж”№еҸҳпјҢзӣҙжҺҘдҪҝеҫ—жҺ§еҲ¶жөҒеҸ‘з”ҹи·іиҪ¬пјҢе№¶ж— зү№еҲ«д№ӢеӨ„гҖӮиҖҢ catch ејӮеёёеҰӮжһңдёәз©әпјҲеҚіеҰӮжһңжҲ‘д»¬з”ҹеҗһејӮеёёпјүпјҢеҲҷејҖй”ҖдёҠзңӢиө·жқҘе’Ңе№іеҮЎ return дёҖж ·гҖӮз„¶иҖҢпјҢnew Exception е®һйҷ…дёҠжҳҜдёҖдёӘйқһеёёжҳӮиҙөзҡ„ж“ҚдҪңгҖӮеӣ дёәејӮеёёеҜ№иұЎеңЁз”ҹжҲҗзҡ„ж—¶еҖҷпјҢе…¶зҲ¶зұ»жһ„йҖ еҮҪж•° Throwable дёӯзҡ„дёҖйғЁеҲҶдјҡи°ғз”Ё fillInStackTrace() ж“ҚдҪңгҖӮиҝҷдёӘ fillInStackTrace() еҮҪж•°пјҢдјҡиҜ•еӣҫжҠҠеҪ“еүҚжҠӣеҮәејӮеёёзҡ„ж Ҳеё§е…ЁйғҪеӣҠжӢ¬еңЁеҶ…пјҢеңЁе®һйҷ…зҡ„иҝҗиЎҢд№ӢдёӯпјҢжңүеҸҜиғҪеҜјиҮҙеӨҚжқӮзҡ„ CPU еҜ„еӯҳеҷЁиҜ»еҶҷж“ҚдҪңгҖӮиҝҷз§ҚиҜ»еҶҷж“ҚдҪңзҡ„еӨҚжқӮеәҰдёҺжҳҜеҗҰдҪҝз”Ё jit ж— е…іпјҢд№ҹе°ұдёҚеҸҜиғҪдёә jit й”ҒдјҳеҢ–пјҢжҳҜдёҖз§ҚеҫҲжҳӮиҙөзҡ„еӣәе®ҡжҲҗжң¬гҖӮ
  [гҖҠJavaиҷҡжӢҹжңәжҳҜеҰӮдҪ•еӨ„зҗҶејӮеёёзҡ„гҖӢ][6]дёӯжҸҗеҲ°зҡ„е®һйӘҢе№¶дёҚд»ЈиЎЁ Java in real world зҡ„е·ҘдҪңзҠ¶еҶөпјҢеӣ дёәзҺ°е®һдёӯеҮ д№ҺжІЎжңүж Ҳж·ұеҸӘдёә1зҡ„ж–№жі•и°ғз”ЁпјҢдёҖдёӘжЎҶжһ¶жҲ–иҖ…е®№еҷЁпјҢжң¬иә«е°ұдјҡеёҰжқҘеҮ еҚҒеұӮзҡ„и°ғз”Ёж Ҳж·ұеәҰгҖӮ

з»“и®ә

еңЁејҖеҗҜJITзҡ„жғ…еҶөдёӢпјҢthrowд№ҹдёҚдјҡеўһеҠ еӨҡе°‘зі»з»ҹејҖй”ҖгҖӮ

  еӣәз„¶жҳҜе®һиҜқгҖӮ

  з„¶иҖҢзҺ°е®һд№ӢдёӯпјҢthrow еҚҙдёҚиғҪзҰ»ејҖд»»дҪ• Throwable зҡ„еӯҗзұ»пјҢжҲ‘д»¬еңЁдҪҝз”ЁејӮеёёжңәеҲ¶зҡ„ж—¶еҖҷпјҢеҝ…йЎ»иғҢиҙҹдёҠз”ҹжҲҗж Ҳеё§иҝҷж ·дёҖдёӘжІүйҮҚзҡ„иҙҹжӢ…пјҢз©әи°Ҳ throw зҡ„жҖ§иғҪдјҳеҢ–жҳҜж— ж„Ҹд№үзҡ„гҖӮжүҖд»Ҙ

жүҖд»ҘеҪ“дҪ йҒҮеҲ°жңүдәәиҜҙtry-catchдёҖе®ҡиҰҒе°‘з”ЁдјҡеҪұе“ҚжҖ§иғҪж—¶пјҢжҲ–и®ёдҪ е°ұдёҚдјҡеҶҚеҺ»зӣІд»Һиҝҷз§ҚвҖңе»әи®®вҖқдәҶгҖӮ

еҚҙжҳҜдёҖз§ҚиҝҮдәҺд№җи§Ӯзҡ„з»“и®әгҖӮжҲ‘д»¬еҪ“з„¶дёҚиғҪж— иҠӮеҲ¶ең°дҪҝз”Ё try-catchпјҢеӣ е®ғдёҚд»…дҪҝзЁӢеәҸеҸҳеҫ—ж”ҜзҰ»з ҙзўҺпјҢиҖҢдё”йҷӨйқһдёҚдјҡеҸ‘з”ҹејӮеёёжҠӣеҮәпјҢеҗҰеҲҷ JVM еҜ№е®ғиҝӣиЎҢзҡ„дјҳеҢ–пјҢеҸӘжҳҜжқҜж°ҙиҪҰи–ӘгҖӮ

  жҲ‘д»¬еә”еҪ“ж°ёиҝңи®°дҪҸпјҢжҠӣеҮәејӮеёёжҳҜжҳӮиҙөзҡ„пјҢдёҚжҳҜеӣ дёә try-catch жҳҜжҳӮиҙөзҡ„пјҢеӣ дёәж— и®әжҖҺд№ҲдҪҝз”ЁејӮеёёпјҢејӮеёёйғҪжҳҜжҳӮиҙөзҡ„гҖӮ

йҷ„еҺҹж–Үпјҡ

зј–з Ғж—¶жҲ‘д»¬еёёеёёиў«иҰҒжұӮе°ҪйҮҸеҮҸе°‘try-catchиҜӯеҸҘеқ—пјҢзҗҶз”ұе°ұжҳҜе°ұз®—дёҚжҠӣејӮеёёе®ғд»¬д№ҹдјҡеҪұе“ҚжҖ§иғҪгҖӮз„¶иҖҢеҪұе“Қз©¶з«ҹжңүеӨҡеӨ§е‘ўпјҹиҜӯеҸҘеқ—еә”иҜҘж”ҫеңЁеҫӘзҺҜдҪ“еҶ…йғЁиҝҳжҳҜеӨ–йғЁе‘ўпјҹдёӢйқўиҜ‘ж–Үе°ҶиҜҰз»ҶйҳҗйҮҠJavaиҷҡжӢҹжңәеӨ„зҗҶејӮеёёзҡ„жңәеҲ¶гҖӮ
иҷҪз„¶ж–ҮдёӯжІЎжңүиҝӣиЎҢжҖ§иғҪеҲҶжһҗпјҢдҪҶж–Үжң«жҸҗдҫӣдәҶдёҖдәӣеҹәеҮҶжөӢиҜ•зҡ„ж–Үз« пјҢе…ҲжҠҠз»“и®әеҶҷеңЁеүҚеӨҙпјҡtry-catchиҜӯеҸҘеқ—еҮ д№ҺдёҚдјҡеҪұе“ҚзЁӢеәҸиҝҗиЎҢжҖ§иғҪпјҒеңЁејҖеҗҜJITзҡ„жғ…еҶөдёӢпјҢthrowд№ҹдёҚдјҡеўһеҠ еӨҡе°‘зі»з»ҹејҖй”ҖгҖӮ

ејӮеёёжңәеҲ¶еҸҜд»Ҙи®©дҪ йЎәеҲ©зҡ„еӨ„зҗҶзЁӢеәҸиҝҗиЎҢиҝҮзЁӢдёӯжүҖйҒҮеҲ°зҡ„и®ёеӨҡж„ҸжғідёҚеҲ°зҡ„жғ…еҶөгҖӮдёәдәҶиҜҙжҳҺJavaиҷҡжӢҹжңәеӨ„зҗҶејӮеёёзҡ„ж–№ејҸпјҢжҲ‘д»¬жқҘзңӢдёҖдёӘеҗҚдёәNitPickyMathзҡ„зұ»пјҢе®ғжҸҗдҫӣдәҶй’ҲеҜ№ж•ҙеһӢзҡ„жұӮжЁЎиҝҗз®—гҖӮе’ҢзӣҙжҺҘиҝӣиЎҢиҝҗз®—ж“ҚдҪңдёҚеҗҢзҡ„жҳҜпјҢиҜҘж–№жі•йҷӨйӣ¶жғ…еҶөдёӢе°ҶжҠӣеҮәеҸ—жЈҖжҹҘзҡ„ејӮеёёпјҲchecked exceptionsпјүгҖӮеңЁJavaиҷҡжӢҹжңәдёӯйҷӨйӣ¶ж—¶еҗҢж ·д№ҹдјҡжҠӣеҮәArithmeticExceptionејӮеёёгҖӮNitPickyMathзұ»жҠӣеҮәзҡ„ејӮеёёе®ҡд№үеҰӮдёӢпјҡ

class DivideByZeroException extends Exception {
}

NitPickyMathзұ»зҡ„remainderж–№жі•з®ҖеҚ•ең°жҚ•иҺ·е№¶жҠӣеҮәдәҶејӮеёёпјҡ

static int remainder(int dividend, int divisor)
throws DivideByZeroException {
try {
return dividend % divisor;
}
catch (ArithmeticException e) {
throw new DivideByZeroException();
}
}

remainderж–№жі•д»…д»…еҸӘжҳҜе°ҶдёӨдёӘintе…ҘеҸӮиҝӣиЎҢдәҶжұӮжЁЎиҝҗз®—пјҲд№ҹдҪҝз”ЁдәҶйҷӨжі•пјүгҖӮеҪ“йҷӨж•°дёә0ж—¶пјҢжұӮжЁЎиҝҗз®—е°ҶжҠӣеҮәArithmeticExceptionејӮеёёпјҢиҜҘж–№жі•е°ҶжҚ•иҺ·иҝҷдёӘејӮеёёе№¶жҠӣеҮәдёҖдёӘиҮӘе®ҡд№үDivideByZeroExceptionејӮеёёгҖӮ

DivideByZeroException е’ҢArithmeticException зҡ„дёҚеҗҢд№ӢеӨ„еңЁдәҺеүҚиҖ…жҳҜеҸ—жЈҖжҹҘејӮеёёпјҢиҖҢеҗҺиҖ…жҳҜйқһеҸ—жЈҖжҹҘејӮеёёгҖӮеӣ жӯӨеҗҺиҖ…жҠӣеҮәж—¶дёҚйңҖиҰҒеңЁж–№жі•еӨҙж·»еҠ throwsиҜӯеҸҘгҖӮErrorжҲ–RuntimeExceptionзұ»зҡ„жүҖжңүеӯҗзұ»йғҪжҳҜйқһеҸ—жЈҖжҹҘејӮеёёпјҲдҫӢеҰӮArithmeticExceptionе°ұжҳҜRuntimeExceptionзҡ„еӯҗзұ»пјүгҖӮ

дҪҝз”ЁjavacеҜ№remainderж–№жі•иҝӣиЎҢзј–иҜ‘пјҢе°Ҷеҫ—еҲ°еҰӮдёӢеӯ—иҠӮз Ғпјҡ

еҸҜд»ҘзңӢеҲ°remainderзҡ„еӯ—иҠӮз ҒеәҸеҲ—дё»иҰҒеҲҶжҲҗдәҶдёӨйғЁеҲҶпјҢз¬¬дёҖйғЁеҲҶжҳҜж–№жі•жӯЈеёёжү§иЎҢзҡ„и·Ҝеҫ„пјҢиҝҷйғЁеҲҶеҜ№еә”зҡ„pcзЁӢеәҸи®Ўж•°еҷЁеҒҸз§»дёә0еҲ°3гҖӮз¬¬дәҢйғЁеҲҶжҳҜcatchиҜӯеҸҘпјҢpcеҒҸз§»дёә4еҲ°12гҖӮ

иҝҗиЎҢж—¶пјҢеӯ—иҠӮз ҒеәҸеҲ—дёӯзҡ„iremжҢҮд»Өе°ҶжҠӣеҮәArithmeticExceptionејӮеёёпјҢиҷҡжӢҹжңәе°Ҷдјҡж №жҚ®ејӮеёёжҹҘиЎЁжқҘжүҫеҲ°еҸҜд»Ҙи·іиҪ¬еҲ°зҡ„catchиҜӯеҸҘдҪҚзҪ®гҖӮжҜҸдёӘеҗ«жңүcatchиҜӯеҸҘзҡ„ж–№жі•зҡ„еӯ—иҠӮз ҒдёӯйғҪйҷ„еёҰдәҶдёҖдёӘејӮеёёиЎЁпјҢе®ғеҢ…еҗ«жҜҸдёӘејӮеёёtryиҜӯеҸҘеқ—зҡ„жқЎзӣ®пјҲentryпјүгҖӮжҜҸдёӘжқЎзӣ®йғҪжңүеӣӣйЎ№дҝЎжҒҜпјҡиө·зӮ№гҖҒз»ҲзӮ№гҖҒи·іиҪ¬зҡ„pcеҒҸз§»дҪҚзҪ®д»ҘеҸҠиҜҘејӮеёёзұ»жүҖеңЁеёёйҮҸжұ дёӯзҡ„зҙўеј•гҖӮremainderж–№жі•зҡ„ејӮеёёиЎЁеҰӮдёӢжүҖзӨәпјҡ

дёҠйқўзҡ„ејӮеёёиЎЁжҳҫзӨәдәҶtryиҜӯеҸҘеқ—зҡ„иө·е§ӢдҪҚзҪ®дёә0пјҢз»“жқҹдҪҚзҪ®дёә4пјҲдёҚеҢ…еҗ«4пјүпјҢеҰӮжһңArithmeticExceptionејӮеёёеңЁ0-3зҡ„иҜӯеҸҘеқ—дёӯжҠӣеҮәпјҢйӮЈд№Ҳpcи®Ўж•°еҷЁе°ҶзӣҙжҺҘи·іиҪ¬еҲ°еҒҸз§»дёә4зҡ„дҪҚзҪ®гҖӮ

еҰӮжһңеңЁиҝҗиЎҢж—¶жҠӣеҮәдәҶдёҖдёӘејӮеёёпјҢйӮЈд№ҲjavaиҷҡжӢҹжңәдјҡжҢүйЎәеәҸжҗңзҙўж•ҙдёӘејӮеёёиЎЁжүҫеҲ°еҢ№й…Қзҡ„жқЎзӣ®пјҢе№¶дё”д»…дјҡеҢ№й…ҚеҲ°еңЁе…¶жҢҮе®ҡиҢғеӣҙеҶ…зҡ„ејӮеёёгҖӮеҪ“жүҫеҲ°з¬¬дёҖдёӘеҢ№й…Қзҡ„жқЎзӣ®еҗҺпјҢиҷҡжӢҹжңәдҫҝе°ҶзЁӢеәҸи®Ўж•°еҷЁи®ҫзҪ®дёәж–°зҡ„еҒҸз§»дҪҚзҪ®пјҢз„¶еҗҺз»§з»ӯжү§иЎҢжҢҮд»ӨгҖӮеҰӮжһңжІЎжңүжқЎзӣ®иў«еҢ№й…ҚеҲ°пјҢjavaиҷҡжӢҹжңәдјҡеј№еҮәеҪ“еүҚзҡ„ж Ҳеё§пјҲеҒңжӯўжү§иЎҢеҪ“еүҚж–№жі•пјүпјҢе№¶з»§з»ӯеҗ‘дёҠпјҲи°ғз”Ёremainderж–№жі•зҡ„ж–№жі•пјүжҠӣеҮәеҗҢж ·зҡ„ејӮеёёгҖӮеҪ“з„¶дёҠзә§ж–№жі•д№ҹдёҚдјҡз»§з»ӯжӯЈеёёжү§иЎҢзҡ„пјҢе®ғеҗҢж ·йңҖиҰҒжҹҘиЎЁжқҘеӨ„зҗҶиҜҘејӮеёёпјҢеҰӮжӯӨеҸҚеӨҚгҖӮ

ејҖеҸ‘иҖ…еҸҜд»ҘдҪҝз”Ёthrowз”іжҳҺжқҘжҠӣеҮәдёҖдёӘејӮеёёпјҢе°ұеғҸremainderж–№жі•зҡ„catchеқ—дёӯйӮЈж ·гҖӮзӣёеә”зҡ„еӯ—иҠӮз ҒжҸҸиҝ°еҰӮдёӢпјҡ
ж“ҚдҪңз Ғ ж“ҚдҪңж•° жҸҸиҝ°
athrow ж—  еј№еҮәThrowableеҜ№иұЎеј•з”ЁпјҢе№¶жҠӣеҮәиҜҘејӮеёё

athrowжҢҮд»Өеј№еҮәж“ҚдҪңж•°ж Ҳж ҲйЎ¶зҡ„еј•з”ЁпјҢиҜҘеј•з”Ёеә”еҪ“дёәThrowableзҡ„еӯҗзұ» (жҲ–иҖ…е°ұжҳҜ ThrowableиҮӘиә«)гҖӮ
жҖқиҖғ

жҲ‘д№ҹдҪҝз”ЁJMHиҝӣиЎҢдәҶжөӢиҜ•пјҢзҺҜеўғе’Ңз»ҶиҠӮе°ұдёҚеҲ—еҮәдәҶгҖӮе…¶дёӯдҪҝз”ЁдәҶ-XintеҸӮж•°жҺ§еҲ¶JITзғӯзӮ№зј–иҜ‘пјҢз»“жһңеҰӮдёӢпјҡ
ејӮеёёжҠӣеҮә е…ій—ӯJIT ејҖеҗҜJIT(й»ҳи®ӨејҖеҗҜ)
Aж— ејӮеёёжҠӣеҮә дёӨиҖ…иҖ—ж—¶еҮ д№ҺзӣёеҗҢ дёӨиҖ…иҖ—ж—¶еҮ д№ҺзӣёеҗҢ
AжҜҸж¬ЎйғҪжҠӣејӮеёё AиҖ—ж—¶зәҰжҳҜBзҡ„30еҖҚ дёӨиҖ…иҖ—ж—¶еҮ д№ҺзӣёеҗҢ

дәҶи§ЈдәҶиҜ‘ж–Үдёӯзҡ„ејӮеёёзҡ„жңәеҲ¶еҗҺпјҢжҲ‘д»¬зҹҘйҒ“try-catchе…¶е®һдёҚиҝҮжҳҜеңЁclassж–Үд»¶дёӯеҠ дәҶдёҖдёӘејӮеёёиЎЁз”ЁдәҺејӮеёёжҹҘиЎЁпјҢеҰӮжһңжІЎжңүејӮеёёжҠӣеҮәпјҢзЁӢеәҸзҡ„жү§иЎҢж–№ејҸе’ҢдёҚеҢ…еҗ«try-catchеқ—е®Ңе…ЁзӣёеҗҢгҖӮеҰӮжһңжңүејӮеёёжҠӣеҮәпјҢйӮЈд№ҲжҖ§иғҪзҡ„зЎ®дјҡдёӢйҷҚпјҢиҖҢиҝҷжҳҜжңүthrowеҜјиҮҙзҡ„пјҢдёҺtry-catchж— е…ігҖӮжӯӨж—¶йңҖиҰҒж №жҚ®е®һйҷ…зҡ„дёҡеҠЎжқҘйў„дј°иҜҘж–№жі•жҠӣеҮәејӮеёёзҡ„йў‘зҺҮжңүеӨҡй«ҳпјҢе°ұз®—дҪ дёҚеҺ»з®ЎпјҢеҪ“ж–№жі•иў«жү§иЎҢж¬Ўж•°иҝҮеӨҡж—¶пјҢjavaиҷҡжӢҹжңәд№ҹдјҡйҖҡиҝҮJITжқҘзј–иҜ‘иҝҷж®өж–№жі•пјҢзј–иҜ‘иҝҮеҗҺдёӨиҖ…зҡ„жү§иЎҢж•ҲзҺҮд№ҹжҳҜеҮ д№ҺзӣёеҗҢзҡ„гҖӮжіЁж„ҸпјҢе…ій—ӯJITеҗҺеҫӘзҺҜж–№жі•ж•ҙдҪ“жҖ§иғҪдёӢйҷҚдәҶеҮ еҚҒеҖҚгҖӮ

жүҖд»ҘеҪ“дҪ йҒҮеҲ°жңүдәәиҜҙtry-catchдёҖе®ҡиҰҒе°‘з”ЁдјҡеҪұе“ҚжҖ§иғҪж—¶пјҢжҲ–и®ёдҪ е°ұдёҚдјҡеҶҚеҺ»зӣІд»Һиҝҷз§ҚвҖңе»әи®®вҖқдәҶгҖӮеҪ“з„¶еңЁзҹҘжҷ“иҝҷдёӘдҝЎжҒҜзҡ„еҗҢж—¶пјҢжҲ‘д»¬еҸҚеҖ’жӣҙеә”иҜҘеҺ»жҖқиҖғеҰӮдҪ•д»ҺдёҡеҠЎе’Ңд»Јз ҒйҖ»иҫ‘зҡ„и§’еәҰжқҘйҖӮеҪ“ең°дҪҝз”Ёtry-catchеҶҷеҮәжӣҙжјӮдә®зҡ„д»Јз ҒгҖӮ
