  1. еҢәеқ—й“ҫжҳҜжҜ”зү№еёҒзҡ„еҹәзЎҖи®ҫж–ҪгҖӮз”ұеҢәеқ—з»„жҲҗй“ҫпјҢжҳҜдёәеҢәеқ—й“ҫгҖӮеҗ„дёӘеҢәеқ—й“ҫзҡ„жҢҒжңүиҖ…д№Ӣй—ҙпјҢжҖ»жҳҜеңЁзҺ©дёҚзЎ®е®ҡйҖүдё»зҡ„жёёжҲҸпјҢжүҖд»Ҙиҝҷе’ҢжүҖжңүдј з»ҹзҡ„еҲҶеёғејҸж•°жҚ®еә“дёҚеҗҢпјҢжҳҜдёҖдёӘеҺ»дёӯеҝғзҡ„ж•°жҚ®еӯҳеӮЁжЁЎејҸгҖӮ
  2. жҜ”зү№еёҒзҡ„еҢәеқ—й“ҫжҳҜ1.0зҡ„зҺ©жі•пјҢд»ҘеӨӘеқҠжҳҜ2.0зҡ„зҺ©жі•гҖӮжңүдәӣдәәи®Өдёә Hyperledger жҳҜ3.0зҡ„зҺ©жі•пјҢиҝҳжңүеҫ…жҖҖз–‘гҖӮ
  3. еҢәеқ—й“ҫдёҠзҡ„иө„дә§пјҢеҸҜд»ҘжҳҜиҮӘеёҰзҡ„пјҲжҜ”зү№еёҒзҪ‘з»ңйҮҢзҡ„жҜ”зү№еёҒпјҢд»ҘеӨӘеқҠзҪ‘з»ңйҮҢзҡ„д»ҘеӨӘеёҒпјүпјҢд№ҹеҸҜд»ҘжҳҜжҷәиғҪеҗҲзәҰе®ҡд№үе’ҢзәҰжқҹзҡ„гҖӮ
  4. жҷәиғҪеҗҲзәҰжҳҜдёӘзңӢиө·жқҘеҫҲзҫҺеҘҪпјҢе®һйҷ…дёҠеҸӘиғҪжү§иЎҢеңЁжІҷзӣ’йҮҢйқўзҡ„дёңиҘҝгҖӮжӣҫз»ҸеңЁжҹҗдёӘIBMзЁӢеәҸе‘ҳзҡ„еҲҶдә«йҮҢзңӢеҲ°иҝҮпјҢHyperledger зҡ„жҷәиғҪеҗҲзәҰжң¬иҙЁдёҠд№ҹжҳҜ GO зЁӢеәҸпјҢжүҖд»ҘзҗҶи®әдёҠеҸҜд»ҘеҒҡдёҖеҲҮдәӢжғ…гҖӮдҪҶзӣ®еүҚжІЎжңүзңӢеҲ°йҷӨдәҶи°ғз”Ёеҗ„з§Қ Shim API д»ҘеӨ–зҡ„д»»дҪ•з”ЁеӨ„гҖӮжҜ”еҰӮпјҢеҰӮжһңжҲ‘д»¬жғіиҰҒз”ЁжҷәиғҪеҗҲзәҰеҸ‘еҮәеҸҰеӨ–дёҖдёӘи°ғз”ЁиҜ·жұӮпјҢи®©зңҹе®һзҡ„зі»з»ҹеҸ‘з”ҹиҪ¬иҙҰпјҢеҰӮдҪ•еҒҡеҲ°пјҹ
  5. еҫҲеӨҡдәәйғҪжңүзғӯжҖқиҖғпјҢдәәзұ»еҶҚд№ҹеӣһдёҚеҲ°жІЎжңүжҜ”зү№еёҒзҡ„ж—¶д»ЈдәҶгҖӮд№ҹжңүеҶ·жҖқиҖғпјҢеҢәеқ—й“ҫзҡ„ж—¶д»ЈиҝҳжңӘеҲ°жқҘгҖӮе°ұзӣ®еүҚиҖҢиЁҖпјҢзҺ°еңЁзҡ„и®Ўз®—жҖ§иғҪзңҹзҡ„дёҚи¶ід»Ҙж”Ҝж’‘зңҹе®һзҡ„иЎҢдёҡжөҒйҮҸпјҢеҸӘиғҪе…»е…»йёЎпјҢиҝҗиҝҗиӮүгҖӮ
  6. д№ҹжңүи§ӮзӮ№и®ӨдёәпјҢе…¬жңүй“ҫжҳҜзңҹе‘ҪйўҳпјҢиҖҢз§Ғжңүй“ҫжҳҜдјӘе‘ҪйўҳгҖӮе°ұзӣ®еүҚзҡ„и§ӮеҜҹзңӢжқҘпјҢдёҚз®ЎжҳҜ Hyperledger иҝҳжҳҜ CordaпјҢдёӯеҝғеҢ–иҠӮзӮ№жҳҜдёҚеҸҜйҒҝе…Қзҡ„гҖӮCorda жҳҜдёәдәҶ Uniqueness Validty зҡ„иҒҢиҙЈеҲҶзҰ»пјҢи®ҫи®ЎдәҶ Notary иҠӮзӮ№гҖӮ Hyperledger еҲҷжҳҜдёәдәҶж”¶йӣҶжүҖжңүиғҢд№ҰеҗҺзҡ„дәӢеҠЎпјҢеҮҶеӨҮеҶҚжҸҗдәӨз»ҷ peer еҶҷе…ҘиҙҰжң¬дёӯгҖӮжҜ«ж— з–‘й—®пјҢиҝҷдёӨз§Қи®ҫи®ЎйғҪеҮәзҺ°дәҶдёӯеҝғеҢ–зҡ„зі»з»ҹдәӨжұҮзӮ№гҖӮR3 еҢәеқ—й“ҫиҒ”зӣҹзҡ„ејҖеҸ‘иҖ…жҳҺзЎ®е‘ҠиҜүжҲ‘пјҢnotary дјҡжҳҜеҪұе“ҚжҖ§иғҪзҡ„еӣ зҙ гҖӮдҪҶжҖ§иғҪиҝҳеңЁе…¶ж¬ЎпјҢжӣҙйҮҚиҰҒзҡ„жҳҜпјҢеңЁиҒ”зӣҹй“ҫжҲ–иҖ…з§Ғй“ҫйҮҢйқўпјҢи°ҒжқҘжҺҢз®ЎиҝҷдёӘ ordererгҖӮеҢәеқ—й“ҫзҡ„е…ұиҜҶе…¬жӯЈжҖ§пјҢеңЁе…¬жңүй“ҫдёҠз”ұйҷҢз”ҹдәәзҹҝе·Ҙзӣёдә’еҲ¶иЎЎе®һзҺ°пјҢеңЁз§Ғжңүй“ҫдёҠпјҢеҚҙжҳҜеҸҜжңүеҸҜж— зҡ„пјҢзңҹзҡ„жІҰдёәдәҶдёҖдёӘе…ұдә«ж•°жҚ®еә“дәҶгҖӮйӮЈд№ҲпјҢиҒ”зӣҹй“ҫеӯҳеңЁзҡ„д»·еҖјжҳҜд»Җд№ҲпјҢејәиҝ«еӨ§е®¶е…ұдә«иҙҰжң¬йҮҢзҡ„жүҖжңүдҝЎжҒҜеҗ—пјҹ
