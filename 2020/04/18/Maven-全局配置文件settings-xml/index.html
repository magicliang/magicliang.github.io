    • зФ®дЇОе≠ШеВ®еТМзЃ°зРЖзђђдЄЙжЦєзЪДз®≥еЃЪеПСеЄГзЙИжЬђзЪДжЮДдїґгАВ
    • ињЩдЇЫжЮДдїґйАЪеЄЄжШѓй°єзЫЃдЊЭиµЦзЪДе§ЦйГ®еЇУжИЦиАЕж°ЖжЮґзЪДз®≥еЃЪзЙИжЬђгАВ
    • е∞ЖзђђдЄЙжЦєзЪД release жЮДдїґеНХзЛђе≠ШеВ®еЬ® thirdParty-release дїУеЇУдЄ≠пЉМеПѓдї•дЄОй°єзЫЃиЗ™еЈ±зЪД release жЮДдїґеИЖеЉАзЃ°зРЖгАВ
    • й°єзЫЃеЬ®жЮДеїЇжЧґпЉМдЉЪдЉШеЕИдїО thirdParty-release дїУеЇУиОЈеПЦжЙАдЊЭиµЦзЪДзђђдЄЙжЦєжЮДдїґзЪДз®≥еЃЪзЙИжЬђгАВ

    • зФ®дЇОе≠ШеВ®еТМзЃ°зРЖзђђдЄЙжЦєзЪДеЉАеПСзЙИжЬђзЪДжЮДдїґгАВ
    • ињЩдЇЫжЮДдїґйАЪеЄЄжШѓй°єзЫЃдЊЭиµЦзЪДе§ЦйГ®еЇУжИЦиАЕж°ЖжЮґзЪДдЄНз®≥еЃЪзЙИжЬђпЉМдЊЛе¶ВдЄАдЇЫе∞ЪжЬ™еПСеЄГж≠£еЉПзЙИжЬђзЪДжЦ∞зЙєжАІжИЦиАЕдњЃе§НгАВ
    • е∞ЖзђђдЄЙжЦєзЪД snapshot жЮДдїґеНХзЛђе≠ШеВ®еЬ® thirdParty-snapshot дїУеЇУдЄ≠пЉМеПѓдї•дЄОй°єзЫЃиЗ™еЈ±зЪД snapshot жЮДдїґеИЖеЉАзЃ°зРЖгАВ
    • й°єзЫЃеЬ®еЉАеПСињЗз®ЛдЄ≠пЉМе¶ВжЮЬйЬАи¶БдљњзФ®зђђдЄЙжЦєзЪДжЬАжЦ∞еЉАеПСзЙИжЬђпЉМеПѓдї•дїО thirdParty-snapshot дїУеЇУиОЈеПЦгАВ
      йАЪињЗе∞ЖжЮДдїґжМЙзЕІз®≥еЃЪжАІеТМжЭ•жЇРињЫи°МеИЖз±їе≠ШеВ®пЉМMaven еПѓдї•жЫіе•љеЬ∞зЃ°зРЖй°єзЫЃзЪДдЊЭиµЦеЕ≥з≥їеТМжЮДеїЇињЗз®Л:
