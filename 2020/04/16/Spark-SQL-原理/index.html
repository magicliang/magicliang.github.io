Spark SQLуџётЈЉт▒ЋтјєуеІ

СИ║С║єу╗ЎуєЪТѓЅуџё RDBMS СйєтЈѕСИЇуљєУДБ MapReduce уџёТіђТю»С║║тЉўТЈљСЙЏт┐ФжђЪСИіТЅІуџётиЦтЁи№╝їHive т║ћУ┐љУђїућЪ№╝їС╗ќТў»тйЊТЌХтћ»СИђУ┐љУАїтюе Hadoop СИіуџёSQL-On-Hadoop тиЦтЁисђѓ СйєТў» MapReduce У«Ау«ЌУ┐ЄуеІСИГтцДжЄЈуџёСИГжЌ┤уБЂуЏўУљйтю░У┐ЄуеІТХѕУђЌС║єтцДжЄЈуџё I/O№╝їжЎЇСйјуџёУ┐љУАїТЋѕујЄ№╝їСИ║С║єТЈљжФў SQL уџёТЅДУАїТЋѕујЄ№╝їтцДжЄЈуџё SQL-On-HadoopтиЦтЁит╝ђтДІС║ДућЪ№╝їУђї Shark Тў»тЁХСИГСИђСИфУАеуј░УЙЃСИ║уфЂтЄ║уџёжА╣уЏ«сђѓ

SharkТў»С╝»тЁІтѕЕт«ъжфїт«ц Spark ућЪТђЂуј»тбЃуџёу╗ёС╗ХС╣ІСИђ№╝їт«ЃСИ╗УдЂС┐«Тћ╣С║єтєЁтГўу«Ауљє№╝їуЅЕуљєУ«АтѕњтњїТЅДУАїСИЅСИфТеАтЮЌ№╝їтђ╝тЙЌт«ЃУЃйУ┐љУАїтюе Spark уџёт╝ЋТЊјСИі№╝їС╗јУђїТЈљжФў SQL ТЪЦУ»буџёТЋѕујЄсђѓ

СйєТў»жџЈуЮђ Spark уџётЈЉт▒Ћ№╝їShark т»╣ Hive У┐ЄтцџуџёСЙЮУхќтѕХу║дС║є Spark уџёУ«ЙУ«Ауљєт┐хтњїтљёСИфу╗ёС╗ХС╣ІжЌ┤уџёуЏИС║њу╗ДТЅ┐№╝їТЅђС╗Ц Spark тЏбжўЪтЂюТГбС║єт»╣ Shark уџёт╝ђтЈЉ№╝їТЈљтЄ║С║є SparkSQL жА╣уЏ«сђѓ тЏаСИ║ТЉєУё▒С║єHive уџёУ┐Єт║дСЙЮУхќ№╝їSpark SQLтюеТЋ░ТЇ«тЁ╝т«╣ТђД№╝їТђДУЃйС╝ўтїќтњїу╗ёС╗ХТЅЕт▒ЋуГЅтљёСИфТќ╣жЮбжЃйтЙЌтѕ░С║єТъЂтцДуџёТќ╣СЙ┐тњїтЈЉт▒Ћсђѓ

ТЈљтЄ║С║є SparkSQL жА╣уЏ«С╣Ітљј№╝їSQL On Spark тЈЉт▒ЋтЄ║С║єСИцТЮАТћ»у║┐№╝їSparkSQL тњїHive on Sparkсђѓ

Spark SQL УДБТъљТхЂуеІ

Spark SQL т»╣ SQL У»ГтЈЦуџётцёуљєтњїтЁ│у│╗тъІТЋ░ТЇ«т║Њу▒╗С╝╝№╝їтЇ│У»ЇТ│Ћ/У»ГТ│ЋУДБТъљ№╝їу╗Љт«џ№╝їС╝ўтїќ№╝їТЅДУАїсђѓ Spark SQL С╝џтЁѕт░є SQL У»ГтЈЦУДБТъљТѕљСИђТБхТаЉ№╝їуёХтљјСй┐ућетљёуДЇУДётѕЎт»╣ Tree У┐ЏУАїу╗Љт«џтњїС╝ўтїќуГЅтцёуљєсђѓ

Сй┐ућеSessionCatalogС┐ЮтГўтЁЃТЋ░ТЇ«

тюеУДБТъљ SQL У»ГтЈЦС╣ІтЅЇ№╝їС╝џтѕЏт╗║ SparkSession№╝їтюе Spark2сђѓ0уЅѕТюгС╣Ітљј№╝їSparkSessionт░ЂУБЁС║є SparkContext тњї SQLContext тѕЏт╗║№╝їС╣ЪСИЇтєЇтї║тѕє SQLContext тњї HiveContextсђѓ

ТХЅтЈітѕ░У»ИтдѓУАетљЇ№╝їтГЌТ«хтљЇуД░тњїтГЌТ«ху▒╗тъІуГЅтЁЃТЋ░ТЇ«жЃйС╝џС┐ЮтГўтюе SessionCatalog СИГсђѓ ТГцтцќ№╝їтѕЏт╗║СИ┤ТЌХУАеТѕќУђЁУ»ЋтЏЙуџёУ┐ЄуеІ№╝їтЁХт«ът░▒С╝џтЙђ SessionCatalog Т│етєїсђѓ

УДБТъљ SQL: Сй┐уће ANTLR ућЪТѕљТюфУДБТъљуџёжђ╗УЙЉУ«Атѕњ

тЈфУдЂТў»тюеТЋ░ТЇ«т║Њу▒╗тъІуџёТіђТю»жЄїжЮб№╝їТ»ћтдѓС╝ау╗Ъуџё MySQL№╝їOracle ТѕќУђЁуј░тюетцДТЋ░ТЇ«жбєтЪЪуџёHiveсђѓ т«ЃуџётЪ║Тюгуџё SQL ТЅДУАїуџёТеАтъІ№╝їжЃйТў»у▒╗С╝╝уџё№╝їждќтЁѕжЃйТў»УдЂућЪТѕљСИђТЮА SQL У»ГтЈЦуџёТЅДУАїУ«Атѕњсђѓ

тйЊУ░Ѓуће SparkSession уџё sql ТѕќУђЁ SQLContext уџё sql Тќ╣Т│ЋТЌХ№╝їт░▒С╝џСй┐уће SparkSqlParser У┐ЏУАїУДБТъљ SQLсђѓ Сй┐ућеуџё ANTLR У┐ЏУАїУ»ЇТ│ЋУДБТъљтњїУ»ГТ│ЋУДБТъљсђѓ т«ЃтѕєСИ║У»ЇТ│ЋтѕєТъљтњїТъёт╗║тѕєТъљТаЉТѕќУђЁУ»ГТ│ЋТаЉ AST 2СИфТГЦжфцТЮЦућЪТѕљUnresolvedLogicalPlanсђѓ

Сй┐ућетѕєТъљтЎе Analyzer УДБТъљжђ╗УЙЉУ«Атѕњ

тюеУ»ЦжўХТ«х№╝їAnalyzer С╝џСй┐уће Analyzer Rules№╝їт╣Ху╗Њтљѕ SessionCatalog№╝їт»╣ТюфУДБТъљуџёжђ╗УЙЉУ«АтѕњУ┐ЏУАїУДБТъљ№╝їућЪТѕљти▓УДБТъљуџёжђ╗УЙЉУ«Атѕњсђѓ

Сй┐ућеС╝ўтїќтЎе Optimizer С╝ўтїќжђ╗УЙЉУ«Атѕњ

С╝ўтїќтЎеС╣ЪТў»С╝џт«џС╣ЅСИђтЦЌ Rules№╝їтѕЕућеУ┐ЎС║Џ Rules т»╣жђ╗УЙЉУ«Атѕњтњї Expression У┐ЏУАїУ┐ГС╗Бтцёуљє№╝їС╗јУђїСй┐тЙЌТаЉуџёУіѓуѓ╣У┐ЏУАїС╝ўтїќсђѓ тюеС╝ау╗Ъуџё Oracle уГЅТЋ░ТЇ«т║ЊСИГ№╝їжђџтИИжЃйС╝џућЪТѕљтцџСИфТЅДУАїУ«Атѕњ№╝їуёХтљјТа╣ТЇ«С╝ўтїќтЎежњѕт»╣тцџСИфУ«АтѕњжђЅТІЕСИђСИфТюђтЦйуџёУ«Атѕњсђѓ УђїSparkSQLСИГ№╝їТў»т»╣СИђСИфти▓ућЪТѕљуџёТЅДУАїУ┐ЏУАїС╝ўтїќућЪТѕљСИђСИфТќ░уџёТЅДУАїУ«Атѕњсђѓ

Т»ћтдѓ№╝їСИђСИф SQL У»ГтЈЦ

ТГцТЌХ№╝їтюеТЅДУАїУ«АтѕњУДБТъљтЄ║ТЮЦуџёТЌХтђЎсђѓ тЁХт«ът░▒Тў»ТїЅуЁДsql тјЪт░ЂСИЇтіеуџёТаитГљТЮЦУДБТъљТѕљтЈ»С╗ЦТЅДУАїуџёУ«Атѕњуџёсђѓ СйєТў»OptimizerС╝џт»╣ТЅДУАїУ«АтѕњУ┐ЏУАїС╝ўтїќсђѓ тюеУ┐ЎСИфСЙІтГљСИГуџё where ТЮАС╗ХТў»тЈ»С╗ЦТћЙтЁЦтГљТЪЦУ»бСИГуџё№╝їУ┐ЎТаитГљТЪЦУ»буџёТЋ░ТЇ«жЄЈтцДтцДтЈўт░Ј№╝їтЈ»С╗ЦС╝ўтїќТЅДУАїжђЪт║дсђѓ уЏИт║ћуџёТЅДУАїУ«Атѕњт░▒С╝џтЈўСИ║

Сй┐ућеSparkPlannerућЪТѕљуЅЕуљєУ«Атѕњ

SparkPlannerСй┐уће Planning Strategies№╝їт»╣С╝ўтїќтљјуџёжђ╗УЙЉУ«АтѕњУ┐ЏУАїУйгТЇб№╝їућЪТѕљтЈ»С╗ЦТЅДУАїуџёуЅЕуљєУ«АтѕњSparkPlanсђѓ

ућЪТѕљуџёуЅЕуљєУ«Атѕњтї║тѕФС║јжђ╗УЙЉУ«АтѕњуџёТў»ТЅДУАїУ«АтѕњтюеУ┐ЎжЄїти▓у╗ЈжЮътИИТИЁТЎ░С║єсђѓ С╗јтЊфСИфТќЄС╗ХУ»╗тЈќС╗ђС╣ѕТЋ░ТЇ«№╝їС╗јтЊфжЄїУ»╗тЈќТЋ░ТЇ«№╝їтдѓСйЋТЊЇСйюуГЅуГЅжЃйти▓у╗ЈТўјуА«С║єсђѓ

Сй┐уће QueryExecution ТЅДУАїуЅЕуљєУ«Атѕњ

ТГцТЌХУ░Ѓуће SparkPlan уџё execute Тќ╣Т│Ћ№╝їт║Ћт▒ѓтЁХт«ъти▓у╗ЈтюеУДдтЈЉтѕЏт╗║т╣ХТЅДУАї Job С║є№╝їуёХтљјУ┐ћтЏъуЏИт║ћу╗ЊТъюсђѓ

СИЅ SparkSQLСИГуЅЕуљєУ«АтѕњуџёТЅДУАї

Spark уџёТа╣тЪ║: RDD

СИјУ«ИтцџСИЊТюЅуџётцДТЋ░ТЇ«тцёуљєт╣│тЈ░СИЇтљї№╝їSpark т╗║уФІтюеу╗ЪСИђТійУ▒Ауџё RDD С╣ІСИі№╝їСй┐тЙЌт«ЃтЈ»С╗ЦС╗ЦтЪ║ТюгСИђУЄ┤уџёТќ╣т╝Јт║ћт»╣СИЇтљїуџётцДТЋ░ТЇ«тцёуљєтю║ТЎ»сђѓ

С╣Ът░▒Тў»У»┤№╝їУдЂуљєУДБSparkSQL№╝їждќтЁѕУдЂуљєУДБ SparkRDDсђѓ

RDDуџётЁеуД░Тў» Resilient Distributed Datasets№╝їТў»СИђСИфт«╣жћЎуџё№╝їт╣ХУАїуџёТЋ░ТЇ«у╗ЊТъё№╝їтЈ»С╗ЦУ«ЕућеТѕиТўЙуц║уџёт░єТЋ░ТЇ«тГўтѓетѕ░уБЂуЏўтњїтєЁтГўСИГсђѓ тљїТЌХ№╝їRDD У┐ўТЈљСЙЏС║єСИђу╗ёСИ░т»їуџёТЊЇСйюТЮЦТЊЇСйюУ┐ЎС║ЏТЋ░ТЇ«сђѓ тюеУ┐ЎС║ЏТЊЇСйюСИГ№╝їУ»Итдѓ map№╝їflatMap№╝їfilter уГЅУйгТЇбТЊЇСйют«ъуј░С║єmonadТеАт╝Ј№╝їтЙѕтЦйтю░тЦЉтљѕС║єScalaуџёжЏєтљѕТЊЇСйюсђѓжЎцТГцС╣Ітцќ№╝їRDDУ┐ўТЈљСЙЏС║єУ»ИтдѓjoinсђЂgroupByсђЂreduceByKeyуГЅТЏ┤СИ║Тќ╣СЙ┐уџёТЊЇСйю№╝ѕТ│еТёЈ№╝їreduceByKeyТў»action№╝їУђїжЮъtransformation№╝Ѕ№╝їС╗ЦТћ»ТїЂтИИУДЂуџёТЋ░ТЇ«У┐љу«Ќсђѓ

RDDТЈљСЙЏС║єСИцТќ╣жЮбуџёуЅ╣ТђДpersistenceтњїpatitioning№╝їућеТѕитЈ»С╗ЦжђџУ┐ЄpersistСИјpatitionByтЄйТЋ░ТЮЦТјДтѕХRDDуџёУ┐ЎСИцСИфТќ╣жЮбсђѓRDDуџётѕєтї║уЅ╣ТђДСИјт╣ХУАїУ«Ау«ЌУЃйтіЏ(RDDт«џС╣ЅС║єparallerizeтЄйТЋ░)№╝їСй┐тЙЌSparkтЈ»С╗ЦТЏ┤тЦйтю░тѕЕућетЈ»С╝Иу╝ЕуџёуАгС╗ХУхёТ║љсђѓУІЦт░єтѕєтї║СИјТїЂС╣ЁтїќС║їУђЁу╗ЊтљѕУхиТЮЦ№╝їт░▒УЃйТЏ┤тіажФўТЋѕтю░тцёуљєТхижЄЈТЋ░ТЇ«сђѓ

ТЌбуёХ RDD Тў» spark job уџёТа╣тЪ║№╝їТЅђТюЅ Spark уџёУ«Ау«ЌжЃйТў»тЪ║С║ј RDD уџёуЏИт║ћTransformation тњї Action ТЊЇСйюсђѓ SparkуџёТЋ┤СИфућЪТђЂу│╗у╗ЪСИјHadoopТў»т«їтЁетЁ╝т«╣уџё,ТЅђС╗Цт»╣С║јHadoopТЅђТћ»ТїЂуџёТќЄС╗Ху▒╗тъІТѕќУђЁТЋ░ТЇ«т║Њу▒╗тъІ,SparkС╣ЪтљїТаиТћ»ТїЂсђѓ SparkТЈљСЙЏС║єтЈ»С╗ЦУ»╗тЈќ Hadoop ТќЄС╗ХуџёТјЦтЈБ№╝їт»╣С║јтцќжЃетГўтѓетѕЏт╗║УђїУеђ№╝їhadoopRDD тЄйТЋ░Тў»ТюђтИИућеуџётюе Spark СИГтѕЏт╗║ RDD уџёТјЦтЈБсђѓ С╣Ът░▒Тў»У»┤ућеТѕитЈ»С╗ЦућеУ┐ЎСИфТјЦтЈБУ»╗тЈќжЏєуЙцСИГуџёТЋ░ТЇ«т╣ХУйгТЇбТѕљуЏИт║ћуџё RDD№╝їуёХтљјТјЦтЁЦ spark У«Ау«Ќт╝ЋТЊјСИГсђѓ

С╗јуЅЕуљєТЅДУАїУ«Атѕњтѕ░ RDD

у╗ЈУ┐Є Spark уџё Catalyst УДБТъљС╝ўтїќтЎет»╣С║ј sql уџётѕєТъљС╣Ітљј№╝їти▓у╗ЈућЪТѕљС║єуЏИт║ћуџёуЅЕуљєТЅДУАїУ«Атѕњ№╝їУђїт»╣С║јти▓у╗ЈтЈ»С╗Цт╝ђтДІТЅДУАїуџё SparkSQLуџё Physical Plan№╝їтЁХТЅДУАїуЏ«ТаЄт░▒Тў»У░Ѓуће action у«ЌтГљС╣Ітљјуџё RDDсђѓ

SparkPlanСИ╗УдЂтїЁтљФтЏЏуДЇТЊЇСйюу▒╗тъІ:

  • BasicOperator
  • Join
  • Aggregate
  • Sort

т»╣С║ј SparkSQL уџёуЅЕуљєТЅДУАїУ«Атѕњ№╝їтЪ║ТюгтЈ»С╗ЦжђџУ┐ЄУ┐ЎтЏЏуДЇТЊЇСйюу▒╗тъІТЮЦт«їТѕљсђѓ Spark С╝џС╗ЦУ┐ЎтЏЏуДЇТЊЇСйюСИ║тЪ║уАђтѕЏт╗║тЄ║уЏИт║ћуџё taskТЮЦтѕєтѕФТЅДУАїсђѓ У┐ЎтЏЏуДЇТЊЇСйюу▒╗тъІуџёУ»ду╗єС╗Іу╗ЇУ┐ЎжЄїСИЇтЂџУ┐ЄтцџжўљУ┐░сђѓ

SparkSQLСИГС╗╗тіАуџёТЅДУАїТеАт╝Ј

тюеС╗Іу╗ЇС║є SparkSQL тдѓСйЋућЪТѕљуЏИт║ћуџёуЅЕуљєТЅДУАїУ«АтѕњС╗ЦтЈіуЅЕуљєТЅДУАїУ«АтѕњТў»тдѓСйЋТЅДУАїС╣Ітљј№╝їSparkSQL С╗╗тіАт░▒У┐ЏтЁЦС║єтЁиСйЊуџёТЅДУАїжўХТ«хсђѓ SparkSQL тЁиСйЊС╗╗тіАуџёТЅДУАїТќ╣т╝ЈтњїТЎ«жђџ Spark С╗╗тіАуџёТЅДУАїТќ╣т╝Јт╣ХТЌати«тѕФсђѓ Spark С╗╗тіАУ┐љУАїТЌХуџёУДњУЅ▓тЈ»С╗ЦтѕєСИ║СИЅуДЇ: driver№╝їexecutor(worker) С╗ЦтЈіcluster managerсђѓ

SparkСИГуџёdriverтЁХт«ътњїyarnСИГApplication MasterуџётіЪУЃйуЏИу▒╗С╝╝сђѓСИ╗УдЂт«їТѕљС╗╗тіАуџёУ░Ѓт║дС╗ЦтЈітњїexecutorтњїcluster managerУ┐ЏУАїтЇЈУ░Ѓсђѓ

тюеYARNСИГ№╝їТ»ЈСИфApplicationт«ъСЙІжЃйТюЅСИђСИфApplication MasterУ┐ЏуеІ№╝їт«ЃТў»Applicationтљ»тіеуџёуггСИђСИфт«╣тЎесђѓт«ЃУ┤ЪУ┤БтњїResourceManagerТЅЊС║цжЂЊ№╝їт╣ХУ»иТ▒ѓУхёТ║љсђѓУјитЈќУхёТ║љС╣ІтљјтЉіУ»ЅNodeManagerСИ║тЁХтљ»тіеcontainerсђѓтдѓТъют»╣ MapReduce Т»ћУЙЃуєЪТѓЅуџёУ»Ю№╝їСИђт«џСИЇС╝џт»╣ Applicaiton Master жЎїућЪсђѓ тюе Spark on Yarn СИГ№╝їС╗╗тіАуџёТЅДУАїТеАт╝ЈтЈ»С╗ЦтѕєСИ║ yarn-client тњї yarn-cluster СИцуДЇсђѓ

С╗јТи▒т▒ѓТгАуџётљФС╣ЅУ«▓№╝їyarn-clusterтњїyarn-clientТеАт╝Јуџётї║тѕФтЁХт«ът░▒Тў»Application MasterУ┐ЏуеІуџётї║тѕФ№╝їyarn-clusterТеАт╝ЈСИІ№╝їdriverУ┐љУАїтюеAM(Application Master)СИГ№╝їт«ЃУ┤ЪУ┤БтљЉYARNућ│У»иУхёТ║љ№╝їт╣ХуЏЉуЮБСйюСИџуџёУ┐љУАїуіХтєхсђѓтйЊућеТѕиТЈљС║цС║єСйюСИџС╣Ітљј№╝їт░▒тЈ»С╗ЦтЁ│ТјЅClient№╝їСйюСИџС╝џу╗Ду╗ГтюеYARNСИіУ┐љУАїсђѓуёХУђїyarn-clusterТеАт╝ЈСИЇжђѓтљѕУ┐љУАїС║цС║њу▒╗тъІуџёСйюСИџсђѓУђїyarn-clientТеАт╝ЈСИІ№╝їApplication MasterС╗ЁС╗ЁтљЉYARNУ»иТ▒ѓexecutor№╝їclientС╝џтњїУ»иТ▒ѓуџёcontainerжђџС┐АТЮЦУ░Ѓт║дС╗ќС╗гтиЦСйю№╝їС╣Ът░▒Тў»У»┤ClientСИЇУЃйуд╗т╝ђсђѓуюІСИІСИІжЮбуџёСИцт╣ЁтЏЙт║ћУ»ЦС╝џТўјуЎй№╝ѕСИітЏЙТў»yarn-clusterТеАт╝Ј№╝їСИІтЏЙТў»yarn-clientТеАт╝Ј№╝Ѕ№╝џ

SparkSQL СИГС╗╗тіАуџётЁиСйЊТЅДУАї

С║єУДБС║є Spark С╗╗тіАуџёТЅДУАїТеАт╝ЈС╣Ітљј№╝їуЏИт║ћуџё job У┐ўСИЇУЃйжЕгСИіт╝ђтДІУ┐љУАїсђѓ У┐ЎТў»тЏаСИ║СИђСИфт«їТЋ┤уџёТЅДУАїУ«АтѕњтЈ»УЃйТ»ћУЙЃтцЇТЮѓ№╝їт«їТЋ┤уџёТЋ░ТЇ«жЄЈС╣ЪТ»ћУЙЃтцД№╝їуЏ┤ТјЦт«їТЋ┤ТЅДУАїСИЇтцфуј░т«ъсђѓ жђџтИИТЃЁтєхСИІ№╝їСИђСИфтѕєтИЃт╝ЈУ«Ау«Ќт╝ЋТЊјжЃйС╝џт░єСИђСИфт«їТЋ┤ job У┐ЏУАїтѕЄтЅ▓тѕєТЅ╣У┐љУАї№╝їspark С╣ЪТў»У┐ЎС╣ѕтЂџуџёсђѓ

тюе Spark СИГ№╝їtask Тў»СИђСИф Job У┐ЏУАїтѕЄтЅ▓тљјУ┐љУАїуџёТюђт░ЈУ┐љу«ЌтЇЋтЁЃ№╝їСИђУѕгТЃЁтєхСИІ№╝їСИђСИф rdd ТюЅтцџт░ЉСИф partition№╝їт░▒С╝џТюЅтцџт░ЉСИф task№╝їтЏаСИ║Т»ЈСИђСИф task тЈфТў»тцёуљєСИђСИф partition СИіуџёТЋ░ТЇ«сђѓ тйЊтюеYARNСИіУ┐љУАїSparkСйюСИџ№╝їТ»ЈСИфSpark executorСйюСИ║СИђСИфYARNт«╣тЎе(container)С╣Ът░▒Тў» executorУ┐љУАїсђѓSparkтЈ»С╗ЦСй┐тЙЌтцџСИфTasksтюетљїСИђСИфт«╣тЎе(container)жЄїжЮбУ┐љУАїсђѓУ┐ЎуДЇТЃЁтєхтюе spark СИГжђџУ┐ЄТаИТЋ░ТЮЦТјДтѕХ№╝їУ┐ЎС╣ЪТў»СИђСИфСИј MapReduce УЙЃтцДСИЇтљїуџётю░Тќ╣сђѓ

Уђї task У┐ЏУАїу╗ётљѕтѕєТЅ╣тљј№╝їжђџтИИуД░СИ║ stageсђѓ С╣Ът░▒Тў»У»┤СИђСИфJobС╝џУбФТІєтѕєСИ║тцџу╗ёTask№╝їТ»Ју╗ёС╗╗тіАУбФуД░СИ║СИђСИфStageт░▒тЃЈMap Stage№╝ї Reduce StageсђѓStageуџётѕњтѕєтюеRDDуџёУ«║ТќЄСИГТюЅУ»ду╗єуџёС╗Іу╗Ї№╝їу«ђтЇЋуџёУ»┤Тў»С╗ЦshuffleтњїresultУ┐ЎСИцуДЇу▒╗тъІТЮЦтѕњтѕєсђѓтюеSparkСИГТюЅСИцу▒╗task№╝їСИђу▒╗Тў»shuffleMapTask№╝їСИђу▒╗Тў»resultTask№╝їуггСИђу▒╗taskуџёУЙЊтЄ║Тў»shuffleТЅђжюђТЋ░ТЇ«№╝їуггС║їу▒╗taskуџёУЙЊтЄ║Тў»result№╝їstageуџётѕњтѕєС╣ЪС╗ЦТГцСИ║СЙЮТЇ«№╝їshuffleС╣ІтЅЇуџёТЅђТюЅтЈўТЇбТў»СИђСИфstage№╝їshuffleС╣ІтљјуџёТЊЇСйюТў»тЈдСИђСИфstageсђѓ

Spark С╝џСИ║СИЇтљїуџё stage С╗ЦтЈіСИЇтљїуџё taskУ«ЙтЦйтЅЇтљјСЙЮУхќ№╝їТЮЦС┐ЮУ»ЂТЋ┤СИф job У┐љУАїуџёТГБуА«ТђДтњїт«їТЋ┤ТђДсђѓ

тйЊТюђтљјСИђСИфresultTask уџёСИіТИИ task тЁежЃеУ┐љУАїт«їС╣ІтљјтЇ│тЈ»т╝ђтДІУ┐љУАї№╝їУ»ЦresultTask уџёу╗ЊТЮЪС╣ЪТёЈтЉ│уЮђ job уџёТѕљтіЪУ┐љУАїсђѓ

УЄ│ТГц№╝їСИђСИф SparkSQL уџёС╗╗тіАС╗јУДБТъљтѕ░ућЪТѕљжђ╗УЙЉУ«Атѕњ№╝їућЪТѕљуЅЕуљєТЅДУАїУ«Атѕњ№╝їтєЇтѕ░тЁиСйЊТЅДУАїУ┐ћтЏъу╗ЊТъюуџёт«їТЋ┤У┐љУАїтјЪуљєС╗Іу╗Їт«їТ»Ћсђѓ
