Spark SQL еҺҹзҗҶ

Spark SQLзҡ„еҸ‘еұ•еҺҶзЁӢ

дёәдәҶз»ҷзҶҹжӮүзҡ„ RDBMS дҪҶеҸҲдёҚзҗҶи§Ј MapReduce зҡ„жҠҖжңҜдәәе‘ҳжҸҗдҫӣеҝ«йҖҹдёҠжүӢзҡ„е·Ҙе…·пјҢHive еә”иҝҗиҖҢз”ҹпјҢд»–жҳҜеҪ“ж—¶е”ҜдёҖиҝҗиЎҢеңЁ Hadoop дёҠзҡ„SQL-On-Hadoop е·Ҙе…·гҖӮ дҪҶжҳҜ MapReduce и®Ўз®—иҝҮзЁӢдёӯеӨ§йҮҸзҡ„дёӯй—ҙзЈҒзӣҳиҗҪең°иҝҮзЁӢж¶ҲиҖ—дәҶеӨ§йҮҸзҡ„ I/OпјҢйҷҚдҪҺзҡ„иҝҗиЎҢж•ҲзҺҮпјҢдёәдәҶжҸҗй«ҳ SQL зҡ„жү§иЎҢж•ҲзҺҮпјҢеӨ§йҮҸзҡ„ SQL-On-Hadoopе·Ҙе…·ејҖе§Ӣдә§з”ҹпјҢиҖҢ Shark жҳҜе…¶дёӯдёҖдёӘиЎЁзҺ°иҫғдёәзӘҒеҮәзҡ„йЎ№зӣ®гҖӮ

SharkжҳҜдјҜе…ӢеҲ©е®һйӘҢе®Ө Spark з”ҹжҖҒзҺҜеўғзҡ„з»„д»¶д№ӢдёҖпјҢе®ғдё»иҰҒдҝ®ж”№дәҶеҶ…еӯҳз®ЎзҗҶпјҢзү©зҗҶи®ЎеҲ’е’Ңжү§иЎҢдёүдёӘжЁЎеқ—пјҢеҖјеҫ—е®ғиғҪиҝҗиЎҢеңЁ Spark зҡ„еј•ж“ҺдёҠпјҢд»ҺиҖҢжҸҗй«ҳ SQL жҹҘиҜўзҡ„ж•ҲзҺҮгҖӮ

дҪҶжҳҜйҡҸзқҖ Spark зҡ„еҸ‘еұ•пјҢShark еҜ№ Hive иҝҮеӨҡзҡ„дҫқиө–еҲ¶зәҰдәҶ Spark зҡ„и®ҫи®ЎзҗҶеҝөе’Ңеҗ„дёӘз»„д»¶д№Ӣй—ҙзҡ„зӣёдә’з»§жүҝпјҢжүҖд»Ҙ Spark еӣўйҳҹеҒңжӯўдәҶеҜ№ Shark зҡ„ејҖеҸ‘пјҢжҸҗеҮәдәҶ SparkSQL йЎ№зӣ®гҖӮ еӣ дёәж‘Ҷи„ұдәҶHive зҡ„иҝҮеәҰдҫқиө–пјҢSpark SQLеңЁж•°жҚ®е…је®№жҖ§пјҢжҖ§иғҪдјҳеҢ–е’Ңз»„д»¶жү©еұ•зӯүеҗ„дёӘж–№йқўйғҪеҫ—еҲ°дәҶжһҒеӨ§зҡ„ж–№дҫҝе’ҢеҸ‘еұ•гҖӮ

жҸҗеҮәдәҶ SparkSQL йЎ№зӣ®д№ӢеҗҺпјҢSQL On Spark еҸ‘еұ•еҮәдәҶдёӨжқЎж”ҜзәҝпјҢSparkSQL е’ҢHive on SparkгҖӮ

Spark SQL и§ЈжһҗжөҒзЁӢ

Spark SQL еҜ№ SQL иҜӯеҸҘзҡ„еӨ„зҗҶе’Ңе…ізі»еһӢж•°жҚ®еә“зұ»дјјпјҢеҚіиҜҚжі•/иҜӯжі•и§ЈжһҗпјҢз»‘е®ҡпјҢдјҳеҢ–пјҢжү§иЎҢгҖӮ Spark SQL дјҡе…Ҳе°Ҷ SQL иҜӯеҸҘи§ЈжһҗжҲҗдёҖжЈөж ‘пјҢз„¶еҗҺдҪҝз”Ёеҗ„з§Қи§„еҲҷеҜ№ Tree иҝӣиЎҢз»‘е®ҡе’ҢдјҳеҢ–зӯүеӨ„зҗҶгҖӮ

дҪҝз”ЁSessionCatalogдҝқеӯҳе…ғж•°жҚ®

еңЁи§Јжһҗ SQL иҜӯеҸҘд№ӢеүҚпјҢдјҡеҲӣе»ә SparkSessionпјҢеңЁ Spark2гҖӮ0зүҲжң¬д№ӢеҗҺпјҢSparkSessionе°ҒиЈ…дәҶ SparkContext е’Ң SQLContext еҲӣе»әпјҢд№ҹдёҚеҶҚеҢәеҲҶ SQLContext е’Ң HiveContextгҖӮ

ж¶үеҸҠеҲ°иҜёеҰӮиЎЁеҗҚпјҢеӯ—ж®өеҗҚз§°е’Ңеӯ—ж®өзұ»еһӢзӯүе…ғж•°жҚ®йғҪдјҡдҝқеӯҳеңЁ SessionCatalog дёӯгҖӮ жӯӨеӨ–пјҢеҲӣе»әдёҙж—¶иЎЁжҲ–иҖ…иҜ•еӣҫзҡ„иҝҮзЁӢпјҢе…¶е®һе°ұдјҡеҫҖ SessionCatalog жіЁеҶҢгҖӮ

и§Јжһҗ SQL: дҪҝз”Ё ANTLR з”ҹжҲҗжңӘи§Јжһҗзҡ„йҖ»иҫ‘и®ЎеҲ’

еҸӘиҰҒжҳҜеңЁж•°жҚ®еә“зұ»еһӢзҡ„жҠҖжңҜйҮҢйқўпјҢжҜ”еҰӮдј з»ҹзҡ„ MySQLпјҢOracle жҲ–иҖ…зҺ°еңЁеӨ§ж•°жҚ®йўҶеҹҹзҡ„HiveгҖӮ е®ғзҡ„еҹәжң¬зҡ„ SQL жү§иЎҢзҡ„жЁЎеһӢпјҢйғҪжҳҜзұ»дјјзҡ„пјҢйҰ–е…ҲйғҪжҳҜиҰҒз”ҹжҲҗдёҖжқЎ SQL иҜӯеҸҘзҡ„жү§иЎҢи®ЎеҲ’гҖӮ

еҪ“и°ғз”Ё SparkSession зҡ„ sql жҲ–иҖ… SQLContext зҡ„ sql ж–№жі•ж—¶пјҢе°ұдјҡдҪҝз”Ё SparkSqlParser иҝӣиЎҢи§Јжһҗ SQLгҖӮ дҪҝз”Ёзҡ„ ANTLR иҝӣиЎҢиҜҚжі•и§Јжһҗе’ҢиҜӯжі•и§ЈжһҗгҖӮ е®ғеҲҶдёәиҜҚжі•еҲҶжһҗе’Ңжһ„е»әеҲҶжһҗж ‘жҲ–иҖ…иҜӯжі•ж ‘ AST 2дёӘжӯҘйӘӨжқҘз”ҹжҲҗUnresolvedLogicalPlanгҖӮ

дҪҝз”ЁеҲҶжһҗеҷЁ Analyzer и§ЈжһҗйҖ»иҫ‘и®ЎеҲ’

еңЁиҜҘйҳ¶ж®өпјҢAnalyzer дјҡдҪҝз”Ё Analyzer RulesпјҢе№¶з»“еҗҲ SessionCatalogпјҢеҜ№жңӘи§Јжһҗзҡ„йҖ»иҫ‘и®ЎеҲ’иҝӣиЎҢи§ЈжһҗпјҢз”ҹжҲҗе·Іи§Јжһҗзҡ„йҖ»иҫ‘и®ЎеҲ’гҖӮ

дҪҝз”ЁдјҳеҢ–еҷЁ Optimizer дјҳеҢ–йҖ»иҫ‘и®ЎеҲ’

дјҳеҢ–еҷЁд№ҹжҳҜдјҡе®ҡд№үдёҖеҘ— RulesпјҢеҲ©з”Ёиҝҷдәӣ Rules еҜ№йҖ»иҫ‘и®ЎеҲ’е’Ң Expression иҝӣиЎҢиҝӯд»ЈеӨ„зҗҶпјҢд»ҺиҖҢдҪҝеҫ—ж ‘зҡ„иҠӮзӮ№иҝӣиЎҢдјҳеҢ–гҖӮ еңЁдј з»ҹзҡ„ Oracle зӯүж•°жҚ®еә“дёӯпјҢйҖҡеёёйғҪдјҡз”ҹжҲҗеӨҡдёӘжү§иЎҢи®ЎеҲ’пјҢз„¶еҗҺж №жҚ®дјҳеҢ–еҷЁй’ҲеҜ№еӨҡдёӘи®ЎеҲ’йҖүжӢ©дёҖдёӘжңҖеҘҪзҡ„и®ЎеҲ’гҖӮ иҖҢSparkSQLдёӯпјҢжҳҜеҜ№дёҖдёӘе·Із”ҹжҲҗзҡ„жү§иЎҢиҝӣиЎҢдјҳеҢ–з”ҹжҲҗдёҖдёӘж–°зҡ„жү§иЎҢи®ЎеҲ’гҖӮ

жҜ”еҰӮпјҢдёҖдёӘ SQL иҜӯеҸҘ

жӯӨж—¶пјҢеңЁжү§иЎҢи®ЎеҲ’и§ЈжһҗеҮәжқҘзҡ„ж—¶еҖҷгҖӮ е…¶е®һе°ұжҳҜжҢүз…§sql еҺҹе°ҒдёҚеҠЁзҡ„ж ·еӯҗжқҘи§ЈжһҗжҲҗеҸҜд»Ҙжү§иЎҢзҡ„и®ЎеҲ’зҡ„гҖӮ дҪҶжҳҜOptimizerдјҡеҜ№жү§иЎҢи®ЎеҲ’иҝӣиЎҢдјҳеҢ–гҖӮ еңЁиҝҷдёӘдҫӢеӯҗдёӯзҡ„ where жқЎд»¶жҳҜеҸҜд»Ҙж”ҫе…ҘеӯҗжҹҘиҜўдёӯзҡ„пјҢиҝҷж ·еӯҗжҹҘиҜўзҡ„ж•°жҚ®йҮҸеӨ§еӨ§еҸҳе°ҸпјҢеҸҜд»ҘдјҳеҢ–жү§иЎҢйҖҹеәҰгҖӮ зӣёеә”зҡ„жү§иЎҢи®ЎеҲ’е°ұдјҡеҸҳдёә

дҪҝз”ЁSparkPlannerз”ҹжҲҗзү©зҗҶи®ЎеҲ’

SparkPlannerдҪҝз”Ё Planning StrategiesпјҢеҜ№дјҳеҢ–еҗҺзҡ„йҖ»иҫ‘и®ЎеҲ’иҝӣиЎҢиҪ¬жҚўпјҢз”ҹжҲҗеҸҜд»Ҙжү§иЎҢзҡ„зү©зҗҶи®ЎеҲ’SparkPlanгҖӮ

з”ҹжҲҗзҡ„зү©зҗҶи®ЎеҲ’еҢәеҲ«дәҺйҖ»иҫ‘и®ЎеҲ’зҡ„жҳҜжү§иЎҢи®ЎеҲ’еңЁиҝҷйҮҢе·Із»Ҹйқһеёёжё…жҷ°дәҶгҖӮ д»Һе“ӘдёӘж–Үд»¶иҜ»еҸ–д»Җд№Ҳж•°жҚ®пјҢд»Һе“ӘйҮҢиҜ»еҸ–ж•°жҚ®пјҢеҰӮдҪ•ж“ҚдҪңзӯүзӯүйғҪе·Із»ҸжҳҺзЎ®дәҶгҖӮ

дҪҝз”Ё QueryExecution жү§иЎҢзү©зҗҶи®ЎеҲ’

жӯӨж—¶и°ғз”Ё SparkPlan зҡ„ execute ж–№жі•пјҢеә•еұӮе…¶е®һе·Із»ҸеңЁи§ҰеҸ‘еҲӣе»әе№¶жү§иЎҢ Job дәҶпјҢз„¶еҗҺиҝ”еӣһзӣёеә”з»“жһңгҖӮ

дёү SparkSQLдёӯзү©зҗҶи®ЎеҲ’зҡ„жү§иЎҢ

Spark зҡ„ж №еҹә: RDD

дёҺи®ёеӨҡдё“жңүзҡ„еӨ§ж•°жҚ®еӨ„зҗҶе№іеҸ°дёҚеҗҢпјҢSpark е»әз«ӢеңЁз»ҹдёҖжҠҪиұЎзҡ„ RDD д№ӢдёҠпјҢдҪҝеҫ—е®ғеҸҜд»Ҙд»Ҙеҹәжң¬дёҖиҮҙзҡ„ж–№ејҸеә”еҜ№дёҚеҗҢзҡ„еӨ§ж•°жҚ®еӨ„зҗҶеңәжҷҜгҖӮ

д№ҹе°ұжҳҜиҜҙпјҢиҰҒзҗҶи§ЈSparkSQLпјҢйҰ–е…ҲиҰҒзҗҶи§Ј SparkRDDгҖӮ

RDDзҡ„е…Ёз§°жҳҜ Resilient Distributed DatasetsпјҢжҳҜдёҖдёӘе®№й”ҷзҡ„пјҢе№¶иЎҢзҡ„ж•°жҚ®з»“жһ„пјҢеҸҜд»Ҙи®©з”ЁжҲ·жҳҫзӨәзҡ„е°Ҷж•°жҚ®еӯҳеӮЁеҲ°зЈҒзӣҳе’ҢеҶ…еӯҳдёӯгҖӮ еҗҢж—¶пјҢRDD иҝҳжҸҗдҫӣдәҶдёҖз»„дё°еҜҢзҡ„ж“ҚдҪңжқҘж“ҚдҪңиҝҷдәӣж•°жҚ®гҖӮ еңЁиҝҷдәӣж“ҚдҪңдёӯпјҢиҜёеҰӮ mapпјҢflatMapпјҢfilter зӯүиҪ¬жҚўж“ҚдҪңе®һзҺ°дәҶmonadжЁЎејҸпјҢеҫҲеҘҪең°еҘ‘еҗҲдәҶScalaзҡ„йӣҶеҗҲж“ҚдҪңгҖӮйҷӨжӯӨд№ӢеӨ–пјҢRDDиҝҳжҸҗдҫӣдәҶиҜёеҰӮjoinгҖҒgroupByгҖҒreduceByKeyзӯүжӣҙдёәж–№дҫҝзҡ„ж“ҚдҪңпјҲжіЁж„ҸпјҢreduceByKeyжҳҜactionпјҢиҖҢйқһtransformationпјүпјҢд»Ҙж”ҜжҢҒеёёи§Ғзҡ„ж•°жҚ®иҝҗз®—гҖӮ

RDDжҸҗдҫӣдәҶдёӨж–№йқўзҡ„зү№жҖ§persistenceе’ҢpatitioningпјҢз”ЁжҲ·еҸҜд»ҘйҖҡиҝҮpersistдёҺpatitionByеҮҪж•°жқҘжҺ§еҲ¶RDDзҡ„иҝҷдёӨдёӘж–№йқўгҖӮRDDзҡ„еҲҶеҢәзү№жҖ§дёҺе№¶иЎҢи®Ўз®—иғҪеҠӣ(RDDе®ҡд№үдәҶparallerizeеҮҪж•°)пјҢдҪҝеҫ—SparkеҸҜд»ҘжӣҙеҘҪең°еҲ©з”ЁеҸҜдјёзј©зҡ„зЎ¬д»¶иө„жәҗгҖӮиӢҘе°ҶеҲҶеҢәдёҺжҢҒд№…еҢ–дәҢиҖ…з»“еҗҲиө·жқҘпјҢе°ұиғҪжӣҙеҠ й«ҳж•Ҳең°еӨ„зҗҶжө·йҮҸж•°жҚ®гҖӮ

ж—ўз„¶ RDD жҳҜ spark job зҡ„ж №еҹәпјҢжүҖжңү Spark зҡ„и®Ўз®—йғҪжҳҜеҹәдәҺ RDD зҡ„зӣёеә”Transformation е’Ң Action ж“ҚдҪңгҖӮ Sparkзҡ„ж•ҙдёӘз”ҹжҖҒзі»з»ҹдёҺHadoopжҳҜе®Ңе…Ёе…је®№зҡ„,жүҖд»ҘеҜ№дәҺHadoopжүҖж”ҜжҢҒзҡ„ж–Үд»¶зұ»еһӢжҲ–иҖ…ж•°жҚ®еә“зұ»еһӢ,Sparkд№ҹеҗҢж ·ж”ҜжҢҒгҖӮ SparkжҸҗдҫӣдәҶеҸҜд»ҘиҜ»еҸ– Hadoop ж–Үд»¶зҡ„жҺҘеҸЈпјҢеҜ№дәҺеӨ–йғЁеӯҳеӮЁеҲӣе»әиҖҢиЁҖпјҢhadoopRDD еҮҪж•°жҳҜжңҖеёёз”Ёзҡ„еңЁ Spark дёӯеҲӣе»ә RDD зҡ„жҺҘеҸЈгҖӮ д№ҹе°ұжҳҜиҜҙз”ЁжҲ·еҸҜд»Ҙз”ЁиҝҷдёӘжҺҘеҸЈиҜ»еҸ–йӣҶзҫӨдёӯзҡ„ж•°жҚ®е№¶иҪ¬жҚўжҲҗзӣёеә”зҡ„ RDDпјҢз„¶еҗҺжҺҘе…Ҙ spark и®Ўз®—еј•ж“ҺдёӯгҖӮ

д»Һзү©зҗҶжү§иЎҢи®ЎеҲ’еҲ° RDD

з»ҸиҝҮ Spark зҡ„ Catalyst и§ЈжһҗдјҳеҢ–еҷЁеҜ№дәҺ sql зҡ„еҲҶжһҗд№ӢеҗҺпјҢе·Із»Ҹз”ҹжҲҗдәҶзӣёеә”зҡ„зү©зҗҶжү§иЎҢи®ЎеҲ’пјҢиҖҢеҜ№дәҺе·Із»ҸеҸҜд»ҘејҖе§Ӣжү§иЎҢзҡ„ SparkSQLзҡ„ Physical PlanпјҢе…¶жү§иЎҢзӣ®ж Үе°ұжҳҜи°ғз”Ё action з®—еӯҗд№ӢеҗҺзҡ„ RDDгҖӮ

SparkPlanдё»иҰҒеҢ…еҗ«еӣӣз§Қж“ҚдҪңзұ»еһӢ:

  • BasicOperator
  • Join
  • Aggregate
  • Sort

еҜ№дәҺ SparkSQL зҡ„зү©зҗҶжү§иЎҢи®ЎеҲ’пјҢеҹәжң¬еҸҜд»ҘйҖҡиҝҮиҝҷеӣӣз§Қж“ҚдҪңзұ»еһӢжқҘе®ҢжҲҗгҖӮ Spark дјҡд»Ҙиҝҷеӣӣз§Қж“ҚдҪңдёәеҹәзЎҖеҲӣе»әеҮәзӣёеә”зҡ„ taskжқҘеҲҶеҲ«жү§иЎҢгҖӮ иҝҷеӣӣз§Қж“ҚдҪңзұ»еһӢзҡ„иҜҰз»Ҷд»Ӣз»ҚиҝҷйҮҢдёҚеҒҡиҝҮеӨҡйҳҗиҝ°гҖӮ

SparkSQLдёӯд»»еҠЎзҡ„жү§иЎҢжЁЎејҸ

еңЁд»Ӣз»ҚдәҶ SparkSQL еҰӮдҪ•з”ҹжҲҗзӣёеә”зҡ„зү©зҗҶжү§иЎҢи®ЎеҲ’д»ҘеҸҠзү©зҗҶжү§иЎҢи®ЎеҲ’жҳҜеҰӮдҪ•жү§иЎҢд№ӢеҗҺпјҢSparkSQL д»»еҠЎе°ұиҝӣе…ҘдәҶе…·дҪ“зҡ„жү§иЎҢйҳ¶ж®өгҖӮ SparkSQL е…·дҪ“д»»еҠЎзҡ„жү§иЎҢж–№ејҸе’Ңжҷ®йҖҡ Spark д»»еҠЎзҡ„жү§иЎҢж–№ејҸе№¶ж— е·®еҲ«гҖӮ Spark д»»еҠЎиҝҗиЎҢж—¶зҡ„и§’иүІеҸҜд»ҘеҲҶдёәдёүз§Қ: driverпјҢexecutor(worker) д»ҘеҸҠcluster managerгҖӮ

Sparkдёӯзҡ„driverе…¶е®һе’ҢyarnдёӯApplication Masterзҡ„еҠҹиғҪзӣёзұ»дјјгҖӮдё»иҰҒе®ҢжҲҗд»»еҠЎзҡ„и°ғеәҰд»ҘеҸҠе’Ңexecutorе’Ңcluster managerиҝӣиЎҢеҚҸи°ғгҖӮ

еңЁYARNдёӯпјҢжҜҸдёӘApplicationе®һдҫӢйғҪжңүдёҖдёӘApplication MasterиҝӣзЁӢпјҢе®ғжҳҜApplicationеҗҜеҠЁзҡ„з¬¬дёҖдёӘе®№еҷЁгҖӮе®ғиҙҹиҙЈе’ҢResourceManagerжү“дәӨйҒ“пјҢе№¶иҜ·жұӮиө„жәҗгҖӮиҺ·еҸ–иө„жәҗд№ӢеҗҺе‘ҠиҜүNodeManagerдёәе…¶еҗҜеҠЁcontainerгҖӮеҰӮжһңеҜ№ MapReduce жҜ”иҫғзҶҹжӮүзҡ„иҜқпјҢдёҖе®ҡдёҚдјҡеҜ№ Applicaiton Master йҷҢз”ҹгҖӮ еңЁ Spark on Yarn дёӯпјҢд»»еҠЎзҡ„жү§иЎҢжЁЎејҸеҸҜд»ҘеҲҶдёә yarn-client е’Ң yarn-cluster дёӨз§ҚгҖӮ

д»Һж·ұеұӮж¬Ўзҡ„еҗ«д№үи®ІпјҢyarn-clusterе’Ңyarn-clientжЁЎејҸзҡ„еҢәеҲ«е…¶е®һе°ұжҳҜApplication MasterиҝӣзЁӢзҡ„еҢәеҲ«пјҢyarn-clusterжЁЎејҸдёӢпјҢdriverиҝҗиЎҢеңЁAM(Application Master)дёӯпјҢе®ғиҙҹиҙЈеҗ‘YARNз”іиҜ·иө„жәҗпјҢе№¶зӣ‘зқЈдҪңдёҡзҡ„иҝҗиЎҢзҠ¶еҶөгҖӮеҪ“з”ЁжҲ·жҸҗдәӨдәҶдҪңдёҡд№ӢеҗҺпјҢе°ұеҸҜд»Ҙе…іжҺүClientпјҢдҪңдёҡдјҡз»§з»ӯеңЁYARNдёҠиҝҗиЎҢгҖӮз„¶иҖҢyarn-clusterжЁЎејҸдёҚйҖӮеҗҲиҝҗиЎҢдәӨдә’зұ»еһӢзҡ„дҪңдёҡгҖӮиҖҢyarn-clientжЁЎејҸдёӢпјҢApplication Masterд»…д»…еҗ‘YARNиҜ·жұӮexecutorпјҢclientдјҡе’ҢиҜ·жұӮзҡ„containerйҖҡдҝЎжқҘи°ғеәҰд»–д»¬е·ҘдҪңпјҢд№ҹе°ұжҳҜиҜҙClientдёҚиғҪзҰ»ејҖгҖӮзңӢдёӢдёӢйқўзҡ„дёӨе№…еӣҫеә”иҜҘдјҡжҳҺзҷҪпјҲдёҠеӣҫжҳҜyarn-clusterжЁЎејҸпјҢдёӢеӣҫжҳҜyarn-clientжЁЎејҸпјүпјҡ

SparkSQL дёӯд»»еҠЎзҡ„е…·дҪ“жү§иЎҢ

дәҶи§ЈдәҶ Spark д»»еҠЎзҡ„жү§иЎҢжЁЎејҸд№ӢеҗҺпјҢзӣёеә”зҡ„ job иҝҳдёҚиғҪй©¬дёҠејҖе§ӢиҝҗиЎҢгҖӮ иҝҷжҳҜеӣ дёәдёҖдёӘе®Ңж•ҙзҡ„жү§иЎҢи®ЎеҲ’еҸҜиғҪжҜ”иҫғеӨҚжқӮпјҢе®Ңж•ҙзҡ„ж•°жҚ®йҮҸд№ҹжҜ”иҫғеӨ§пјҢзӣҙжҺҘе®Ңж•ҙжү§иЎҢдёҚеӨӘзҺ°е®һгҖӮ йҖҡеёёжғ…еҶөдёӢпјҢдёҖдёӘеҲҶеёғејҸи®Ўз®—еј•ж“ҺйғҪдјҡе°ҶдёҖдёӘе®Ңж•ҙ job иҝӣиЎҢеҲҮеүІеҲҶжү№иҝҗиЎҢпјҢspark д№ҹжҳҜиҝҷд№ҲеҒҡзҡ„гҖӮ

еңЁ Spark дёӯпјҢtask жҳҜдёҖдёӘ Job иҝӣиЎҢеҲҮеүІеҗҺиҝҗиЎҢзҡ„жңҖе°Ҹиҝҗз®—еҚ•е…ғпјҢдёҖиҲ¬жғ…еҶөдёӢпјҢдёҖдёӘ rdd жңүеӨҡе°‘дёӘ partitionпјҢе°ұдјҡжңүеӨҡе°‘дёӘ taskпјҢеӣ дёәжҜҸдёҖдёӘ task еҸӘжҳҜеӨ„зҗҶдёҖдёӘ partition дёҠзҡ„ж•°жҚ®гҖӮ еҪ“еңЁYARNдёҠиҝҗиЎҢSparkдҪңдёҡпјҢжҜҸдёӘSpark executorдҪңдёәдёҖдёӘYARNе®№еҷЁ(container)д№ҹе°ұжҳҜ executorиҝҗиЎҢгҖӮSparkеҸҜд»ҘдҪҝеҫ—еӨҡдёӘTasksеңЁеҗҢдёҖдёӘе®№еҷЁ(container)йҮҢйқўиҝҗиЎҢгҖӮиҝҷз§Қжғ…еҶөеңЁ spark дёӯйҖҡиҝҮж ёж•°жқҘжҺ§еҲ¶пјҢиҝҷд№ҹжҳҜдёҖдёӘдёҺ MapReduce иҫғеӨ§дёҚеҗҢзҡ„ең°ж–№гҖӮ

иҖҢ task иҝӣиЎҢз»„еҗҲеҲҶжү№еҗҺпјҢйҖҡеёёз§°дёә stageгҖӮ д№ҹе°ұжҳҜиҜҙдёҖдёӘJobдјҡиў«жӢҶеҲҶдёәеӨҡз»„TaskпјҢжҜҸз»„д»»еҠЎиў«з§°дёәдёҖдёӘStageе°ұеғҸMap StageпјҢ Reduce StageгҖӮStageзҡ„еҲ’еҲҶеңЁRDDзҡ„и®әж–ҮдёӯжңүиҜҰз»Ҷзҡ„д»Ӣз»ҚпјҢз®ҖеҚ•зҡ„иҜҙжҳҜд»Ҙshuffleе’ҢresultиҝҷдёӨз§Қзұ»еһӢжқҘеҲ’еҲҶгҖӮеңЁSparkдёӯжңүдёӨзұ»taskпјҢдёҖзұ»жҳҜshuffleMapTaskпјҢдёҖзұ»жҳҜresultTaskпјҢз¬¬дёҖзұ»taskзҡ„иҫ“еҮәжҳҜshuffleжүҖйңҖж•°жҚ®пјҢз¬¬дәҢзұ»taskзҡ„иҫ“еҮәжҳҜresultпјҢstageзҡ„еҲ’еҲҶд№ҹд»ҘжӯӨдёәдҫқжҚ®пјҢshuffleд№ӢеүҚзҡ„жүҖжңүеҸҳжҚўжҳҜдёҖдёӘstageпјҢshuffleд№ӢеҗҺзҡ„ж“ҚдҪңжҳҜеҸҰдёҖдёӘstageгҖӮ

Spark дјҡдёәдёҚеҗҢзҡ„ stage д»ҘеҸҠдёҚеҗҢзҡ„ taskи®ҫеҘҪеүҚеҗҺдҫқиө–пјҢжқҘдҝқиҜҒж•ҙдёӘ job иҝҗиЎҢзҡ„жӯЈзЎ®жҖ§е’Ңе®Ңж•ҙжҖ§гҖӮ

еҪ“жңҖеҗҺдёҖдёӘresultTask зҡ„дёҠжёё task е…ЁйғЁиҝҗиЎҢе®Ңд№ӢеҗҺеҚіеҸҜејҖе§ӢиҝҗиЎҢпјҢиҜҘresultTask зҡ„з»“жқҹд№ҹж„Ҹе‘ізқҖ job зҡ„жҲҗеҠҹиҝҗиЎҢгҖӮ

иҮіжӯӨпјҢдёҖдёӘ SparkSQL зҡ„д»»еҠЎд»Һи§ЈжһҗеҲ°з”ҹжҲҗйҖ»иҫ‘и®ЎеҲ’пјҢз”ҹжҲҗзү©зҗҶжү§иЎҢи®ЎеҲ’пјҢеҶҚеҲ°е…·дҪ“жү§иЎҢиҝ”еӣһз»“жһңзҡ„е®Ңж•ҙиҝҗиЎҢеҺҹзҗҶд»Ӣз»Қе®ҢжҜ•гҖӮ
