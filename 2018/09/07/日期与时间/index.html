JSR 310 Java DateдёҺTime API

ж–°ж—§ API зҡ„жӣҙиҝӯ

ж—§зҡ„ Java API дё»иҰҒеҢ…жӢ¬java.util.Dateе’Ңjava.util.Calendar дёӨдёӘеҢ…зҡ„еҶ…е®№гҖӮиҝҷдёӨдёӘеҢ…зҡ„ж—¶й—ҙзұ»еһӢжҳҜеҸҜеҸҳзҡ„гҖӮеҰӮ Date зҡ„е®һдҫӢеҸҜд»ҘйҖҡиҝҮ setYear жқҘдә§з”ҹеҸҳеҢ–гҖӮ

JSR 310 дёӯеҢ…жӢ¬зҡ„ж—Ҙжңҹзұ»еһӢдё»иҰҒжңүпјҡ

  1. и®Ўз®—жңәж—¶й—ҙпјҡInstantпјҢеҜ№еә” java.util.DateпјҢе®ғд»ЈиЎЁдәҶдёҖдёӘзЎ®е®ҡзҡ„ж—¶й—ҙзӮ№пјҢеҚізӣёеҜ№дәҺж ҮеҮҶJavaзәӘе…ғпјҲ1970е№ҙ1жңҲ1ж—Ҙпјүзҡ„еҒҸз§»йҮҸпјӣдҪҶдёҺjava.util.Dateзұ»дёҚеҗҢзҡ„жҳҜе…¶зІҫзЎ®еҲ°дәҶзәіз§’зә§еҲ«гҖӮ
  2. дәәзұ»ж—¶й—ҙпјҡеҜ№еә”дәҺдәәзұ»иҮӘиә«зҡ„и§ӮеҝөпјҢжҜ”еҰӮLocalDateе’ҢLocalTimeгҖӮд»–д»¬д»ЈиЎЁдәҶдёҖиҲ¬зҡ„ж—¶еҢәжҰӮеҝөпјҢиҰҒд№ҲжҳҜж—ҘжңҹпјҲдёҚеҢ…еҗ«ж—¶й—ҙпјүпјҢиҰҒд№ҲжҳҜж—¶й—ҙпјҲдёҚеҢ…еҗ«ж—ҘжңҹпјүпјҢзұ»дјјдәҺjava.sqlзҡ„иЎЁзӨәж–№ејҸгҖӮжӯӨеӨ–пјҢиҝҳжңүдёҖдёӘMonthDayпјҢе®ғеҸҜд»ҘеӯҳеӮЁжҹҗдәәзҡ„з”ҹж—ҘпјҲдёҚеҢ…еҗ«е№ҙд»ҪпјүгҖӮжҜҸдёӘзұ»йғҪеңЁеҶ…йғЁеӯҳеӮЁжӯЈзЎ®зҡ„ж•°жҚ®иҖҢдёҚжҳҜеғҸjava.util.DateйӮЈж ·еҲ©з”ЁеҚҲеӨң12зӮ№жқҘеҢәеҲҶж—ҘжңҹпјҢеҲ©з”Ё1970-01-01жқҘиЎЁзӨәж—¶й—ҙгҖӮиҝҷдәӣзұ»еһӢзҡ„е®һдҫӢжҳҜ immutable зҡ„пјҢиҖҢдё”еҸӘиғҪйҖҡиҝҮе·ҘеҺӮж–№жі•еҲӣе»әгҖӮ

ж—¶еҢәпјҲжҜ”еҰӮEurope/ParisдёҺAmerica/New_Yorkпјүе’Ңи·қUTCзҡ„еҒҸз§»йҮҸпјҲжҜ”еҰӮ+01:00дёҺ-08:00пјүд№Ӣй—ҙзҡ„е·®еҲ«

еҒҸз§»йҮҸд»…д»…жҳҜ UTC пјҲеҹәзЎҖеҒҸз§»иө·зӮ№пјүе’Ңжң¬ең°ж—¶й—ҙд№Ӣй—ҙзҡ„е·®еҖјгҖӮ

иҖҢж—¶еҢәжҳҜдёҖдёӘе…·еҗҚзҡ„и§„еҲҷйӣҶеҗҲпјҢжҸҸиҝ°дәҶеҒҸз§»йҮҸж”№еҰӮдҪ•йҡҸзқҖж—¶й—ҙзҡ„еҸҳеҢ–иҖҢеҸҳеҢ–гҖӮжҜ”еҰӮиҜҙпјҢж—¶еҢәдјҡжҸҸиҝ°дёҖдёӘзү№е®ҡзҡ„еҢәеҹҹпјҲжҜ”еҰӮзәҪзәҰпјүеңЁз»ҷе®ҡзҡ„дёҖдёӘж—¶еҲ»е…·жңүжҹҗдёӘеҒҸз§»йҮҸпјҢд№ӢеҗҺе…·жңүеҸҰдёҖдёӘ еҒҸз§»йҮҸпјҲеңЁжң¬ең°ж—¶й—ҙзәҝдёҠеҲӣе»әдёҖдёӘй—ҙйҡҷжҲ–жҳҜйҮҚеҸ пјҢеҰӮжҳҘз§Ӣж—¶еҲ¶зҡ„еҸҳжҚўпјүгҖӮжҚўиЁҖд№ӢпјҢж—¶еҢәжҳҜжҜ”еҒҸз§»йҮҸжӣҙеӨҚжқӮзҡ„и§„еҲҷйӣҶеҗҲгҖӮ

жңүдёүдёӘзә§еҲ«зҡ„зұ»ж”ҜжҢҒиҝҷдәӣжҰӮеҝөпјҡ

  1. LocalDateTime ж— йңҖдҪҝз”ЁеҒҸз§»йҮҸе’Ңж—¶еҢәе°ұиғҪиЎЁзӨәж—¶й—ҙгҖӮ
  2. OffsetDateTime йўқеӨ–ең°жҢҮе®ҡдәҶеҒҸз§»йҮҸгҖӮ
  3. ZonedDateTime еҲҷеўһеҠ дәҶж—¶еҢәи§„еҲҷгҖӮ

иҝҮеҺ»пјҢеҫҲеӨҡеә”з”ЁйғҪе–ңж¬ўдҪҝз”Ёж—¶еҢәпјҢдҪҶд»–д»¬зңҹжӯЈйңҖиҰҒзҡ„е…¶е®һеҸӘжҳҜеҒҸз§»йҮҸиҖҢе·ІпјҲдҪҝз”ЁеҒҸз§»йҮҸжӣҙз®ҖеҚ•гҖҒжӣҙеҝ«дё”дёҚжҳ“еҮәй”ҷпјүгҖӮжүҖд»ҘжҲ‘д»¬жӣҙеҠ еә”иҜҘдҪҝз”ЁеҒҸз§»йҮҸгҖӮXML Schemaи§„иҢғе°ұжҳҜдёҖдёӘе…ёеһӢпјҢе®ғеҸӘж”ҜжҢҒеҒҸз§»йҮҸиҖҢдёҚж”ҜжҢҒж—¶еҢәгҖӮJSR 310еҸҜд»ҘжҳҺзЎ®иЎЁзӨәеҮәиҝҷдәӣе·®еҲ«гҖӮ

ж—¶еҢәи§„еҲҷдјҡйҡҸзқҖж—¶й—ҙзҡ„жҺЁз§»иҖҢдёҚж–ӯеҸ‘з”ҹеҸҳеҢ–гҖӮе°ұеңЁеҚғзҰ§е№ҙд№ӢеүҚпјҢдёҖдәӣеӣҪе®¶е°Ҷж—¶еҢәз”ұеӣҪйҷ…ж—Ҙз•Ңзәҝд№ӢеҗҺж”№дёәд№ӢеүҚпјӣжӯӨеӨ–пјҢеӨҸж—¶еҲ¶д№ҹеңЁдёҚж–ӯеҸҳеҢ–гҖӮ

е…¶д»–йҷ„еұһзұ»еһӢ

еҸҜд»ҘдҪҝз”ЁDurationз•Ңе®ҡд»»ж„ҸдёӨдёӘInstantд№Ӣй—ҙзҡ„иҢғеӣҙгҖӮ

зҺ°еңЁжңүдёҖдәӣе…·дҪ“зҡ„жҰӮеҝөжқҘиЎЁзӨәYearMonthе’ҢMonthDayпјҢеңЁйҖӮеҪ“зҡ„ж—¶еҖҷеә”иҜҘдҪҝз”ЁиҝҷдёӨдёӘзұ»гҖӮиҝҳжңүдёҖдёӘPeriodзұ»жқҘиЎЁзӨәд»»ж„Ҹзҡ„ж—¶й—ҙе‘ЁжңҹпјҢеҰӮвҖңдёӨе№ҙгҖҒ3дёӘжңҲгҖҒ7еӨ©гҖҒ4е°Ҹж—¶гҖҒ50еҲҶй’ҹвҖқзӯүгҖӮ

ж ёеҝғж—ҘеҺҶжҳҜISOChronologyпјҢй»ҳи®Өжғ…еҶөдёӢдҪҝз”Ёе®ғжқҘжҳ е°„ж—¶й—ҙпјҢе°ұеғҸзӣ®еүҚJava APIдёӯзҡ„GregorianCalendarдёҖж ·гҖӮз„¶иҖҢпјҢжҲ‘д»¬еҜ№е…¶д»–дёҖдәӣе№ҙд»Јд№ҹжҸҗдҫӣдәҶж”ҜжҢҒпјҢеҰӮCopticChronologyе’ҢThaiBuddhistChronologyгҖӮ

MySQL зҡ„ж—¶й—ҙ API жңүеҮ дёӘйҖҡз”Ёзҡ„и®ҫи®ЎиҖғйҮҸпјҡ

  2. еҜ№дәҺж—ҘжңҹпјҢMySQL еҖҫеҗ‘дәҺдҪҝз”Ёе№ҙжңҲж—Ҙзҡ„йЎәеәҸпјҲвҖҷ98-09-04вҖҷпјүпјҢиҖҢдёҚжҳҜе…¶д»–йЎәеәҸгҖӮ
  3. еҸӘжңүдёӨдҪҚж•°еӯ—пјҲtwo-digitпјүзҡ„е№ҙд»ҪеҸҜиғҪеӯҳеңЁдәҢд№үжҖ§пјҢMySQL зҡ„иҪ¬д№үи§„еҲҷжҳҜпјҡ

DATE зұ»еһӢжІЎжңүж—¶й—ҙйғЁеҲҶпјҢйҖҡз”Ёж јејҸдёә вҖҳYYYY-MM-DDвҖҷпјҢеҗҲзҗҶзҡ„еҸ–еҖјиҢғеӣҙдёәвҖҷ1000-01-01вҖҷ иҮі вҖҳ9999-12-31вҖҷгҖӮ

DATETIME зұ»еһӢж—ўжңүж—ҘжңҹйғЁеҲҶпјҢеҸҲжңүж—¶й—ҙйғЁеҲҶгҖӮеҗҲзҗҶзҡ„еҸ–еҖјиҢғеӣҙдёәвҖҷ1000-01-01вҖҷ иҮі вҖҳ9999-12-31вҖҷгҖӮиҝҷзұ»дјј Java 8 зҡ„ LocalDatetimeпјҢжҳҜеұһдәҺдәәзҡ„еҪўејҸпјҢдёҚйңҖиҰҒзҗҶи§Јж—¶еҢәе’ҢеҒҸз§»гҖӮ

TIMESTAMP зұ»еһӢж—ўжңүж—ҘжңҹйғЁеҲҶпјҢеҸҲжңүж—¶й—ҙйғЁеҲҶгҖӮеҗҲзҗҶзҡ„еҸ–еҖјиҢғеӣҙдёә вҖҳ1970-01-01 00:00:01вҖҷ UTC иҮі вҖҳ2038-01-19 03:14:07вҖҷ UTCгҖӮд№ҹе°ұжҳҜиҜҙпјҢ TIMESTAMP иҝҷдёӘзұ»еһӢзұ»дјј Unix Epoch timestampпјҢж—¶й—ҙиҢғеӣҙеҸҲзӘ„пјҢеҹәзЎҖеҒҸз§»йҮҸеҸҲеҹәдәҺ UTCгҖӮ

иҝҷдёӨз§Қзұ»еһӢзҡ„еӯ—йқўйҮҸйҮҢйғҪжҳҜжІЎжңү T зҡ„гҖӮ

DATETIME е’Ң TIMESTAMP еҸҜд»ҘеҢ…еҗ«жңҖеӨҡеҲ°еҫ®з§’е…ӯдҪҚзІҫеәҰзҡ„е°Ҹж•°йғЁеҲҶгҖӮ

еҰӮжһңеҠ е…ҘдәҶе°Ҹж•°йғЁеҲҶпјҡ

DATETIME зҡ„еҸ–еҖјиҢғеӣҙеҸҳдёә вҖҳ1000-01-01 00:00:00.000000вҖҷ еҲ° вҖҳ9999-12-31 23:59:59.999999вҖҷгҖӮ

TIMESTAMP зҡ„еҸ–еҖјиҢғеӣҙеҸҳдёә вҖҳ1970-01-01 00:00:01.000000вҖҷ еҲ° вҖҳ2038-01-19 03:14:07.999999вҖҷгҖӮ

DATETIME е’Ң TIMESTAMP зҡ„иҮӘеҠЁеҲқе§ӢеҢ–е’Ңжӣҙж–°иЎҢдёә

еҸҜд»Ҙд»ҺеҲқе§ӢеҖје’Ңе…¶д»– column иў«дҝ®ж”№ж—¶иҮӘеҠЁдҝ®ж”№еҖјдёӨдёӘз»ҙеәҰжқҘи®ҫзҪ®зұ»еһӢпјҡ

дҪҶжҖ»дҪ“жқҘи®ІдёҚжҳҜеҫҲеҘҪз”ЁгҖӮиҝҳжҳҜжҠҠзјәзңҒеҲқе§ӢеҖји®ҫзҪ®дёәжҹҗдёӘж—¶й—ҙжҲіжҜ”иҫғз®ҖеҚ•гҖӮ

е…·дҪ“ж–ҮжЎЈеңЁгҖҠAutomatic Initialization and Updating for TIMESTAMP and DATETIMEгҖӢгҖӮ

DATETIME е’Ң TIMESTAMP зҡ„е…¶д»–еҢәеҲ«

  1. TIMESTAMP ж¶ҲиҖ—зҡ„еӯҳеӮЁз©әй—ҙжӣҙе°ҸпјҢеӣ дёәе®ғиЎЁиҫҫзҡ„ж—¶й—ҙиҢғеӣҙжӣҙе°Ҹ-жүҖд»ҘиҝҷеҜјиҮҙжңӘжқҘзҡ„ж—¶й—ҙй—®йўҳгҖӮзҫҺеӣўеҫҲеӨҡдәәдёҚеңЁд№ҺиҝҷдёҖзӮ№пјҢж„ҝж„Ҹз”ЁпјҢжҲ‘иҮӘе·ұдёҚеӨӘж„ҝж„Ҹз”ЁпјҢдҪҝз”Ё DATETIME еӨ§ж–№дёҖзӮ№д№ҹжІЎй”ҷгҖӮ
  2. TIMESTAMP дјҡйҡҗи—ҸдёҖдәӣж—¶еҢәзҡ„иҮӘеҠЁиҪ¬жҚўпјҢиҫ“еҮәз»“жһңдјҡеҸ—жңҚеҠЎеҷЁж—¶еҢәй…ҚзҪ®пјҲlocaleпјүеҪұе“ҚгҖӮиҖҢдёҺж—¶еҢәж— е…ізҡ„ DATETIME еҸҚиҖҢеҸҜд»Ҙи®©ж—¶еҢәзҡ„иҪ¬жҚўжҳҫејҸиў«дёҡеҠЎжҺ§еҲ¶гҖӮ
  3. DATETIME еңЁдҪҺзүҲжң¬йҮҢдёҚиғҪдҪҝз”ЁеҮҪж•°з”ҹжҲҗзјәзңҒеҖјпјҲеӣ дёәиҝҷдёӘ postпјүгҖӮиҰҒи°Ёж…Һең°дҪҝз”Ёеӯ—йқўйҮҸдҪңдёәж—¶й—ҙзұ»еһӢеҲ—зҡ„зјәзңҒеҖјпјҢеҰӮжһңйҖүй”ҷдәҶпјҢдјҡеҜјиҮҙжҸ’е…ҘеҮәй”ҷпјҲз”Ё null д№ҹеҫҲеҚұйҷ©пјүпјҡ

ж—¶еҢәзҡ„еҪұе“Қ

TIMESTAMP зұ»еһӢеңЁиў«еӯҳеӮЁж—¶пјҢдјҡиў«иҪ¬еҫҖ UTC ж—¶еҢәпјҲжӯӨеӨ„ UTC иў«еҪ“еҒҡж—¶еҢәжқҘз”ЁпјүпјҢеңЁеҸ–еҖјзҡ„ж—¶еҖҷпјҢеҸҲдјҡиў«еҸ–жҲҗеҪ“еүҚж—¶еҢәзҡ„ж—¶й—ҙ-жүҖд»Ҙ TIMESTAMP еҸҜд»ҘйҖӮеә”жңҚеҠЎеҷЁгҖҒиҝһжҺҘзҡ„ж—¶еҢәеҸҳжӣҙпјҢDATETIMEдёҚиЎҢгҖӮ

иҖҢ MySQL зҡ„ж—¶еҢәи®ҫзҪ®пјҢжҳҜй’ҲеҜ№жҜҸдёҖдёӘиҝһжҺҘеҸҜд»ҘеҚ•зӢ¬и®ҫзҪ®зҡ„гҖӮиҝҷд№ҹе°ұж„Ҹе‘ізқҖпјҢеҰӮжһңж—¶еҢәдёҚжҳҜдёҖдёӘеёёйҮҸпјҢдёҖдёӘ TIMESTAMP иў«еӯҳиҝӣеҺ»дёҖдёӘеҖјпјҢеҸ–еҮәжқҘжҳҜеҸҰдёҖдёӘеҖјгҖӮиҝҷдёӘеҸҜиғҪеӯҳеңЁзҡ„й—®йўҳпјҢжҳҜеӨ§е®¶йғҪе–ңж¬ўз”Ё datetime зҡ„еҺҹеӣ гҖӮж—¶еҢәзӣёе…ізҡ„е…·дҪ“иЎҢдёәпјҢеҸҜд»ҘйҖҡиҝҮ time_zone иҝҷдёӘзҺҜеўғеҸҳйҮҸжқҘдҝ®ж”№гҖӮ

зҹ«жӯЈеҲ°йӣ¶еҖј

еҜ№дәҺдёҚеҗҲзҗҶзҡ„иөӢеҖјпјҢMySQL еҸҜиғҪжҠҠе®ғиөӢеҖјдёәеҗ„з§Қзұ»еһӢзҡ„йӣ¶еҖјпјҢиҝҳеҸҜиғҪдә§з”ҹиӢҘе№ІиӯҰе‘ҠгҖӮ

зІҫеәҰе®ҡд№ү

иҜӯжі•жҳҜ type_name(fsp)гҖӮtype_name еҸҜд»ҘжҳҜ TIME, DATETIME, жҲ–иҖ… TIMESTAMPпјҢиҖҢ fsp жҳҜе°Ҹж•°з§’зІҫеәҰгҖӮ

дёҖдёӘдҫӢеӯҗпјҡ

еҰӮжһңе®ҡд№үйҮҢйқўдёҚеӯҳеңЁз§’ж•°пјҢеӯҳеҖјзҡ„ж—¶еҖҷеҸҜиғҪдјҡиў«жҠ№еҺ»пјҢжҜ”еҖјзҡ„ж—¶еҖҷеҸҜиғҪеӨұиҙҘгҖӮ

ж—Ҙжңҹе’Ңж—¶й—ҙзұ»еһӢзҡ„иҪ¬жҚў

иҝҷдёӨз§Қзұ»еһӢзӣёдә’иҪ¬жҚўжңүзӮ№зұ»дјјз§»дҪҚгҖӮиғҪиЎҘйӣ¶еҲҷиЎҘйӣ¶пјҢеҗҰеҲҷдјҡеҸҳжҲҗе…Ёйӣ¶еҖјгҖӮ
