scoop 安装

首先要有一个 Powershell 3.0 与 .Net 4.5 以上的环境。

在 PowerShell 中输入：

iex (new-object net.webclient).downloadstring('https://get.scoop.sh')

如果没有网络问题，则可以直接安装成功（可能需要修改一些与 admin 有关的 policy）。如果安装不成功，则删除C:\Users\LC\scoop再试一次。

# 安装 curl
scoop install curl
