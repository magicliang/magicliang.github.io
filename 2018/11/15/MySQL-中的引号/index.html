MySQL 中的引号

Created2018-11-15|Updated2026-01-24
|Word Count:58|Reading Time:1mins|Post Views:

标准的 SQL 中只允许用单引号表达字符串类型。有些 SQL 方言允许使用双引号包裹字符串，如 MySQL，有些则不允许，如 Oracle。

反引号是专门用来表达 identifier 的。
