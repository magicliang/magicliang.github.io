ж ҮзӯҫпјҲз©әж јеҲҶйҡ”пјүпјҡ Redis

redis-server

еёёз”Ёзҡ„й…ҚзҪ®ж–Үд»¶еҶ…е®№пјҡ

  • port дёҖиҲ¬жҳҜ 6379
  • logfile ж—Ҙеҝ—ж–Үд»¶пјҲredis зҡ„ж—Ҙеҝ—ж–Үд»¶е’Ң Kafka дёҚдёҖж ·пјҢе’ҢеӯҳеӮЁж–Үд»¶жҳҜеҲҶзҰ»зҡ„пјү
  • dir Redis е·ҘдҪңзӣ®еҪ•пјҲеӯҳж”ҫжҢҒд№…еҢ–е’Ңж—Ҙеҝ—ж–Үд»¶пјү
  • daemonize жҳҜеҗҰд»Ҙе®ҲжҠӨиҝӣзЁӢзҡ„ж–№ејҸжқҘеҗҜеҠЁ Redis

redis зҡ„ minor зүҲжң¬еҸ·еҰӮжһңжҳҜеҘҮж•°пјҢеҲҷеҗ«жңүе®һйӘҢжҖ§ feature зҡ„йқһзЁіе®ҡзүҲжң¬пјӣеҰӮжһңжҳҜеҒ¶ж•°пјҢеҲҷжҳҜзЁіе®ҡзүҲжң¬гҖӮжүҖд»ҘжҲ‘д»¬еә”иҜҘеңЁз”ҹдә§зҺҜеўғдҪҝз”ЁеҒ¶ж•°зүҲжң¬пјҢиҖҢеңЁе®һйӘҢжҖ§зҺҜеўғйҮҢдҪҝз”ЁеҘҮж•°зүҲжң¬гҖӮ
