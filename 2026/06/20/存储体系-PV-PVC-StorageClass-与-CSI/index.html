еӯҳеӮЁдҪ“зі»пјҡPVгҖҒPVCгҖҒStorageClass дёҺ CSI

еүҚеҮ зҜҮе»әз«ӢдәҶи®Ўз®—пјҲPod и°ғеәҰе’ҢиҝҗиЎҢпјүе’ҢзҪ‘з»ңпјҲPod й—ҙйҖҡдҝЎе’ҢжңҚеҠЎеҸ‘зҺ°пјүзҡ„жЁЎеһӢгҖӮжңүзҠ¶жҖҒеә”з”ЁпјҲж•°жҚ®еә“гҖҒж¶ҲжҒҜйҳҹеҲ—гҖҒеҜ№иұЎеӯҳеӮЁпјүйңҖиҰҒжҢҒд№…еҢ–еӯҳеӮЁпјҢPod йҮҚеҗҜеҗҺж•°жҚ®дёҚдёўеӨұгҖӮKubernetes йҖҡиҝҮ PV/PVC/StorageClass дёүеұӮжҠҪиұЎи§ЈиҖҰдәҶеӯҳеӮЁзҡ„дҫӣеә”е’ҢдҪҝз”ЁгҖӮ

еңЁиҷҡжӢҹжңәж—¶д»ЈпјҢеӯҳеӮЁйҖҡеёёз”ұиҝҗз»ҙе·ҘзЁӢеёҲжүӢеҠЁжҢӮиҪҪпјҡNFS зӣ®еҪ•гҖҒSAN LUNгҖҒжң¬ең°зЈҒзӣҳпјҢжҢӮиҪҪи·Ҝеҫ„зЎ¬зј–з ҒеңЁйғЁзҪІи„ҡжң¬йҮҢгҖӮе®№еҷЁеҢ–д№ӢеҗҺпјҢPod йҡҸж—¶еҸҜиғҪжјӮз§»еҲ°дёҚеҗҢиҠӮзӮ№пјҢеӯҳеӮЁе’Ңи®Ўз®—зҡ„з”ҹе‘Ҫе‘Ёжңҹеҝ…йЎ»жҳҫејҸи§ЈиҖҰгҖӮKubernetes зҡ„еӣһзӯ”жҳҜеј•е…ҘдёүдёӘеұӮж¬ЎпјҡPVпјҲе®һйҷ…еӯҳеӮЁзҡ„жҠҪиұЎпјүгҖҒPVCпјҲеә”з”Ёзҡ„еӯҳеӮЁйңҖжұӮеЈ°жҳҺпјүгҖҒStorageClassпјҲеҠЁжҖҒдҫӣеә”зҡ„зӯ–з•Ҙе·ҘеҺӮпјүгҖӮPV/PVC зҡ„з»‘е®ҡжңәеҲ¶е’Ң CSI й©ұеҠЁеҰӮдҪ•и®©еӨ–йғЁеӯҳеӮЁзі»з»ҹжҲҗдёә K8s зҡ„дёҖзӯүе…¬ж°‘пјҢжҳҜжң¬зҜҮзҡ„ж ёеҝғеҶ…е®№гҖӮ

CSIпјҲContainer Storage Interfaceпјүзҡ„еҮәзҺ°и§ЈеҶідәҶ in-tree й©ұеҠЁзҡ„еҺҶеҸІеҢ…иўұгҖӮжӯӨеүҚпјҢжҜҸдёӘдә‘еҺӮе•ҶгҖҒжҜҸдёӘеӯҳеӮЁзі»з»ҹйғҪиҰҒеҗ‘ Kubernetes дё»д»Јз Ғеә“жҸҗдәӨй©ұеҠЁд»Јз ҒпјҢеҚҮзә§и·Ҝеҫ„ж··д№ұдё”е‘Ёжңҹжј«й•ҝгҖӮCSI е®ҡд№үдәҶдёҖеҘ— gRPC жҺҘеҸЈпјҢд»»дҪ•еӯҳеӮЁеҺӮе•ҶйғҪеҸҜд»Ҙзј–еҶҷзӢ¬з«Ӣзҡ„й©ұеҠЁзЁӢеәҸпјҢд»Ҙ DaemonSet + Deployment зҡ„еҪўејҸйғЁзҪІеҲ°йӣҶзҫӨпјҢе®һзҺ°дёҺ Kubernetes зҡ„и§ЈиҖҰгҖӮ

дё»ж•°жҚ®жөҒ

ж•ҙдёӘжөҒзЁӢеҲҶдёӨдёӘйҳ¶ж®өпјҡжҺ§еҲ¶йқўйҳ¶ж®өпјҲCreateVolume + PV еҲӣе»әпјүеҸ‘з”ҹеңЁдә‘/еӯҳеӮЁ API еұӮпјҢдёҺе…·дҪ“иҠӮзӮ№ж— е…іпјӣиҠӮзӮ№йқўйҳ¶ж®өпјҲNodeStageVolume + NodePublishVolumeпјүеҸ‘з”ҹеңЁ Pod иў«и°ғеәҰеҲ°зҡ„иҠӮзӮ№дёҠпјҢиҙҹиҙЈжҠҠеқ—и®ҫеӨҮжҲ–ж–Үд»¶зі»з»ҹе®һйҷ…жҢӮиҝӣе®№еҷЁгҖӮ

ж ёеҝғеҜ№иұЎдёҺж•°жҚ®жөҒ

PV з”ҹе‘Ҫе‘Ёжңҹ

PV жҳҜйӣҶзҫӨзә§еҲ«зҡ„еҜ№иұЎпјҢжҸҸиҝ°дёҖеқ—е®һйҷ…еӯҳеӮЁзҡ„дҪҚзҪ®гҖҒе®№йҮҸе’Ңи®ҝй—®иғҪеҠӣгҖӮз”ҹе‘Ҫе‘ЁжңҹзҠ¶жҖҒжңәпјҡ

  • AvailableпјҡPV е·ІеӯҳеңЁпјҢзӯүеҫ…иў« PVC з»‘е®ҡгҖӮ
  • BoundпјҡPV дёҺжҹҗдёӘ PVC е»әз«ӢдәҶз»‘е®ҡе…ізі»пјҢе…¶д»– PVC ж— жі•еҶҚз»‘е®ҡиҝҷдёӘ PVгҖӮ
  • ReleasedпјҡдёҺд№Ӣз»‘е®ҡзҡ„ PVC иў«еҲ йҷӨпјҢPV иҝӣе…ҘжӯӨзҠ¶жҖҒзӯүеҫ…еӣһж”¶зӯ–з•ҘеӨ„зҗҶгҖӮ
  • еӣһж”¶зӯ–з•ҘдёүйҖүдёҖпјҡRetainпјҲдҝқз•ҷж•°жҚ®пјҢйңҖз®ЎзҗҶе‘ҳжүӢеҠЁжё…зҗҶпјүгҖҒDeleteпјҲеҲ йҷӨеә•еұӮеӯҳеӮЁиө„дә§пјүгҖҒRecycleпјҲе·ІеәҹејғпјҢж—©жңҹз”Ё scrub жё…з©әж•°жҚ®еҗҺеҶҚж¬Ў AvailableпјүгҖӮ

Released зҠ¶жҖҒзҡ„ PV дёҚдјҡиҮӘеҠЁиў«ж–° PVC з»‘е®ҡпјҢеҚідҫҝе®№йҮҸе’Ңи®ҝй—®жЁЎејҸе®Ңе…ЁеҢ№й…ҚгҖӮиҝҷжҳҜдёәдәҶйҳІжӯўж®Ӣз•ҷж•°жҚ®иў«еҸҰдёҖдёӘеә”з”Ёж„ҸеӨ–иҜ»еҸ–гҖӮз®ЎзҗҶе‘ҳзЎ®и®Өж•°жҚ®еӨ„зҗҶе®ҢжҜ•еҗҺпјҢйңҖиҰҒжүӢеҠЁжё…йҷӨ PV зҡ„ claimRef еӯ—ж®өпјҢPV жүҚеӣһеҲ° AvailableгҖӮ

PVC з»‘е®ҡпјҡдёүйҮҚеҢ№й…Қ

PVC еҲӣе»әеҗҺпјҢжҺ§еҲ¶йқўеңЁжүҖжңү Available PV дёӯеҜ»жүҫж»Ўи¶іжқЎд»¶зҡ„еҜ№иұЎпјҢдёүдёӘз»ҙеәҰзјәдёҖдёҚеҸҜпјҡ

з¬¬дёҖпјҢи®ҝй—®жЁЎејҸпјҲAccessModesпјүеҝ…йЎ»е…је®№гҖӮPV еЈ°жҳҺе®ғж”ҜжҢҒе“ӘдәӣжЁЎејҸпјҢPVC еЈ°жҳҺе®ғйңҖиҰҒе“ӘдәӣжЁЎејҸпјҡ

  • RWOпјҲReadWriteOnceпјүпјҡеҗҢдёҖж—¶еҲ»еҸӘжңүдёҖдёӘиҠӮзӮ№еҸҜд»ҘиҜ»еҶҷпјҢеқ—еӯҳеӮЁзҡ„е…ёеһӢжЁЎејҸгҖӮ
  • ROXпјҲReadOnlyManyпјүпјҡеӨҡиҠӮзӮ№еҸӘиҜ»пјҢйҖӮеҗҲйқҷжҖҒиө„дә§еңәжҷҜгҖӮ
  • RWXпјҲReadWriteManyпјүпјҡеӨҡиҠӮзӮ№еҗҢж—¶иҜ»еҶҷпјҢйңҖиҰҒзҪ‘з»ңж–Үд»¶зі»з»ҹпјҲNFSгҖҒCephFSгҖҒAzure Files зӯүпјүж”ҜжҢҒгҖӮ
  • RWOPпјҲReadWriteOncePodпјүпјҡKubernetes 1.22+ еј•е…ҘпјҢе°ҶеҚ•еҶҷйҷҗеҲ¶зІҫз»ҶеҲ° Pod зІ’еәҰпјҢиҖҢйқһиҠӮзӮ№зІ’еәҰгҖӮ

з¬¬дәҢпјҢеӯҳеӮЁйҮҸгҖӮPVC еЈ°жҳҺ requests.storageпјҢPV жҸҗдҫӣ capacity.storageпјҢPV е®№йҮҸеҝ…йЎ»еӨ§дәҺзӯүдәҺ PVC иҜ·жұӮйҮҸгҖӮKubernetes йҖүжӢ©ж»Ўи¶іжқЎд»¶зҡ„жңҖе°Ҹ PVпјҲbest-fit зӯ–з•ҘпјүпјҢеҮҸе°‘иө„жәҗжөӘиҙ№гҖӮ

з¬¬дёүпјҢStorageClassгҖӮPVC йҖҡиҝҮ storageClassName еӯ—ж®өеј•з”ЁпјҢеҝ…йЎ»дёҺ PV зҡ„ storageClassName еҢ№й…ҚпјҢжҲ–иҖ…дёӨиҖ…йғҪдёәз©әпјҲйқҷжҖҒдҫӣз»ҷж—¶еҸҜзңҒз•ҘпјүгҖӮ

StorageClass дёҺеҠЁжҖҒдҫӣз»ҷ

йқҷжҖҒдҫӣз»ҷиҰҒжұӮз®ЎзҗҶе‘ҳйў„е…ҲеҲӣе»ә PVпјҢи§„жЁЎеӨ§ж—¶дёҚзҺ°е®һгҖӮStorageClass еј•е…ҘеҠЁжҖҒдҫӣз»ҷпјҡеҪ“ PVC еј•з”ЁжҹҗдёӘ StorageClass дё”жІЎжңүзҺ°жҲҗзҡ„ Available PV еҸҜеҢ№й…Қж—¶пјҢKubernetes и§ҰеҸ‘иҜҘ StorageClass зҡ„ provisioner еҠЁжҖҒеҲӣе»ә PVгҖӮ

StorageClass е…ій”®еӯ—ж®өпјҡ

  • provisionerпјҡй©ұеҠЁеҗҚз§°пјҢдҫӢеҰӮ ebs.csi.aws.comгҖҒdisk.csi.azure.comгҖӮ
  • parametersпјҡдј йҖ’з»ҷ provisioner зҡ„еҸӮж•°пјҢдҫӢеҰӮзЈҒзӣҳзұ»еһӢпјҲgp3гҖҒpremium-ssdпјүгҖҒIOPSгҖҒеҠ еҜҶй…ҚзҪ®зӯүгҖӮ
  • reclaimPolicyпјҡеҠЁжҖҒеҲӣе»әзҡ„ PV й»ҳи®ӨдҪҝз”ЁжӯӨеӣһж”¶зӯ–з•ҘпјҢйҖҡеёёи®ҫдёә DeleteгҖӮ
  • volumeBindingModeпјҡImmediateпјҲз«ӢеҚідҫӣз»ҷпјүжҲ– WaitForFirstConsumerпјҲе»¶иҝҹеҲ° Pod и°ғеәҰеҗҺдҫӣз»ҷпјҢйҒҝе…Қ Pod е’Ң PV еңЁдёҚеҗҢеҸҜз”ЁеҢәпјүгҖӮ

WaitForFirstConsumer жҳҜйҮҚиҰҒзҡ„жңҖдҪіе®һи·өгҖӮиӢҘи®ҫдёә ImmediateпјҢPV еңЁ PVC еҲӣе»әж—¶з«ӢеҚіеңЁжҹҗдёӘеҸҜз”ЁеҢәеҲӣе»әпјҢдҪҶ Pod еҸҜиғҪиў«и°ғеәҰеҲ°еҸҰдёҖдёӘеҸҜз”ЁеҢәпјҢеҜјиҮҙеқ—еӯҳеӮЁж— жі•и·ЁеҢәжҢӮиҪҪиҖҢеӨұиҙҘгҖӮ

CSI жһ¶жһ„

CSI й©ұеҠЁз”ұдёӨдёӘз»„д»¶з»„жҲҗпјҢйғЁзҪІж–№ејҸдёҚеҗҢпјҢиҒҢиҙЈжҳҺзЎ®еҲҶзҰ»гҖӮ

CSI Controller Plugin йҖҡеёёд»Ҙ Deployment еҪўејҸиҝҗиЎҢпјҢйқ иҝ‘жҺ§еҲ¶е№ійқўпјҲдҪҶдёҚеңЁжҺ§еҲ¶е№ійқўеҶ…пјүгҖӮе®ғе®һзҺ° Controller жңҚеҠЎжҺҘеҸЈпјҡ

  • CreateVolume / DeleteVolumeпјҡи°ғз”ЁеӯҳеӮЁеҗҺз«Ҝ API еҲӣе»ә/еҲ йҷӨеӯҳеӮЁеҚ·гҖӮ
  • ControllerPublishVolume / ControllerUnpublishVolumeпјҡе°ҶеҚ·йҷ„еҠ еҲ°жҹҗдёӘиҠӮзӮ№пјҲеҜ№еә” AWS EBS AttachгҖҒAzure Disk Attach ж“ҚдҪңпјүгҖӮ
  • CreateSnapshot / DeleteSnapshotпјҡеҸҜйҖүиғҪеҠӣпјҢж”ҜжҢҒеҝ«з…§еҠҹиғҪгҖӮ

CSI Node Plugin д»Ҙ DaemonSet еҪўејҸиҝҗиЎҢеңЁжҜҸдёӘиҠӮзӮ№дёҠпјҢе®һзҺ° Node жңҚеҠЎжҺҘеҸЈпјҡ

  • NodeStageVolumeпјҡе°Ҷеқ—и®ҫеӨҮжҢӮиҪҪеҲ°иҠӮзӮ№е…ЁеұҖи·Ҝеҫ„пјҲstaging pathпјүпјҢйҖҡеёёеңЁжӯӨе®ҢжҲҗж–Үд»¶зі»з»ҹж јејҸеҢ–е’ҢеҲқж¬ЎжҢӮиҪҪгҖӮдёҖдёӘеҚ·еңЁеҗҢдёҖиҠӮзӮ№дёҠеҸӘжү§иЎҢдёҖж¬Ў stageгҖӮ
  • NodePublishVolumeпјҡйҖҡиҝҮ bind mount е°Ҷ staging path жҳ е°„еҲ° Pod дё“еұһзҡ„ mount pathгҖӮеҗҢдёҖдёӘеҚ·еҸҜд»Ҙ publish з»ҷеҗҢдёҖиҠӮзӮ№дёҠзҡ„еӨҡдёӘ PodпјҲROX еңәжҷҜпјүгҖӮ
  • NodeUnpublishVolume / NodeUnstageVolumeпјҡеҚёиҪҪзҡ„йҖҶеҗ‘ж“ҚдҪңгҖӮ

VolumeAttachment

VolumeAttachment жҳҜдёҖдёӘйӣҶзҫӨзә§ API еҜ№иұЎпјҲstorage.k8s.io/v1пјүпјҢи®°еҪ•"жҹҗдёӘ PV йҷ„еҠ еҲ°жҹҗдёӘиҠӮзӮ№"иҝҷдёӘдәӢе®һгҖӮе®ғз”ұ attach-detach controller еҲӣе»әпјҢCSI external-attacher sidecar зӣ‘еҗ¬иҝҷдёӘеҜ№иұЎе№¶и°ғз”Ё ControllerPublishVolumeгҖӮVolumeAttachment зҡ„еӯҳеңЁи®© kubelet еҸҜд»ҘеҲӨж–ӯеқ—и®ҫеӨҮжҳҜеҗҰе·Іе°ұз»ӘпјҢеҶҚеҶіе®ҡжҳҜеҗҰз»§з»ӯжү§иЎҢ NodeStageVolumeгҖӮ

sidecar е®№еҷЁжҳҜ CSI й©ұеҠЁи®ҫи®Ўзҡ„е…ій”®жЁЎејҸгҖӮexternal-provisioner зӣ‘еҗ¬ PVC еҸҳжӣҙпјҢexternal-attacher зӣ‘еҗ¬ VolumeAttachmentпјҢnode-driver-registrar еңЁ kubelet жҸ’д»¶зӣ®еҪ•жіЁеҶҢй©ұеҠЁзҡ„ Unix socketгҖӮCSI й©ұеҠЁжң¬иә«еҸӘйңҖе®һзҺ° gRPC жҺҘеҸЈпјҢдёҚйңҖиҰҒзҹҘйҒ“еҰӮдҪ•дёҺ Kubernetes API дәӨдә’пјҢеӨҚжқӮжҖ§з”ұиҝҷдәӣе…¬е…ұ sidecar еҲҶжӢ…гҖӮ
