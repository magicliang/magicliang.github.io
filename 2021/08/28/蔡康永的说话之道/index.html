  1. 阅读本来是赏心悦目的事情。
  2. 透过说话，懂得把别人放在心上，这就是我相信的、蔡康永的说话之道。
  3. 做自己和没礼貌，常在一线之间。刺耳并不等于简单的“直”。
  4. 把话练好，是最划算的事情。把话练好，就是练练把人放在心上。
  5. 你有没有能力想象听你讲话的人是什么心情，想听到什么。
  6. 练习说话很方便。说话其实就关乎你和别人的想法的交流。单单研究话是没有用的，研究人和人的想法有用。
  7. 我们心里其实藏了很多，我们自己都没法搞清楚的事情。
  8. 我们应当尽量把我们相信的事，和我们说的话变为一体。
  9. 能够打动别人的话，包含了很多“生命能量”，而不只是巧言令色，别有居心。
  10. 梦想拥有天使的外表，不如修炼一颗天使的心。与其嫉恶如仇，不如隐恶扬善。

已经很讨人喜欢的你，在未来会更讨人喜欢

  1. 转述第三方的赞美，更可信。

你说什么样的话，你就是什么样的人。

  1. 人把自己内心深处的想法隐藏起来，只会让自己错乱、分裂。
  2. 说话和想法的因果有时候会反直觉。
  3. 不要简单责备人，修复问题的效果不一定好。

外表好不好看，绝对不是人生的决胜点

  1. 讨不讨人喜欢，还比较重要一点。甚至超越一般的、基于能力的量化评价标准。
  2. 别人并不是为了伺候你而存在的。要提供一个可选的解决方案，再去拒绝别人。

沉默没问题的，沉默很正常的

  1. 人在乱放音乐的时候的放松是最自然的。
  2. 为了防止尴尬，大人们喜欢在聊天时放大音量的电视，让电视的声音填补尴尬的空白。
  3. 不要逼自己找话题。

把无谓的胜利让给别人，懂得认输的人很懂说话

  1. 任何强势的胜负，都是无意义的。无损原则，可以一笑置之。
  2. 侃侃而谈，可以很有想法，但不要说到对方哑口无言。哑口无言，意味着别人还有话说不出来。
  3. 人缘差的人，没法打团队战。
  4. 智者说过：每个人都是自己小领土的国王。
  5. 情侣之间，维持爱是最重要的。胜负没有任何意义。

把对方看在眼里，放在心里

  1. 最好三不五时地，带着情感看着对方，让对方感觉两个人之间有暧昧的电流在流动。
  2. 约会就是约会，要含情脉脉。
  3. 并不是懂美食美酒，就是有品位的人。
  4. 应该“以你为尊”，而不是“吃喝为尊”。
  5. 被重视是关键词，重要的是“视”。

不会 Game Over，让人接得下去的话

  1. 遇上对方提起了一个你完全不想接的话题，不必急着要抵抗，而是轻巧地把对方热衷的话题连接到一个很生活的方向就行了。
  2. “比赛会赢吗？”“你会下多大的注？”“你以前的女朋友会怎样想？”

话题卡住了，就换话题，不要恋战

  1. 谈话是发生在当下的事情，快速转换话题或者结束话题都是好的。
  2. 不要坚持拿没有子弹的枪，去敲敌人的头。
  3. 谈话是发生在当下的事。

问的问题越具体，回答的人越省力

  1. 有选项：封闭式。
  2. 有退路：所有问题的答案都有话可讲。
  3. 我们应当看待交往带有目的这件事。交谈轻松、愉快就行了。

聊天时，每个人都想聊自己

  1. 当你自己想要被别人喜欢的时候，你只要把别人放在你自己的位置上来想，那就轮到你来扮演这个“最上道”的朋友了。
  2. 人和人相处得最如胶似漆的时候，也是活在各自的世界里，而不是彼此的世界里的。
  3. 最专心听你讲话的人，是你“最上道”的朋友。
  4. 多说你，少说我。谁也不是陪酒的牛郎。

问题很尖锐，可以倒推回去两三步

  1. 人生难免出现尖锐的问题，逃得过就逃，这是我们的天性。但逃不过的话，就处理吧。
  2. 不要问“你是不是在吸毒”，要问“你最近是不是过得很痛苦”。
  3. 不要理所当然地带有优越感地去跟别人谈话。
  4. 尖锐的问题，可以引入“假设的第三方”。
  5. 内容里措辞变多了，可以软化一些细节。

适度地挑衅，能让谈话热络

  1. 人很敏锐，能听得出谁在敷衍谁。跟在敷衍自己的人谈话，就好像对着墙壁练习挥网球拍一样。
  2. 提要求和接要求，和价值观有关系，不同的价值观需要的价码是不一样的。

不想交浅言深的话，应该避开地雷

  1. 交情深，谈得深。这两个条件，缺一不可。
  2. 不要随便刺痛别人最脆弱的地方。
  3. 谈话工作者的工作内容就就是交浅言深，踩地雷。但这是为了表演效果。
  4. 财务状况、健康状况、感情状况、孩子的成绩，肯恩是别人的苦衷。政治立场可能是别人的强硬坚持这都很容易出争执。

硬生生地报出数字，很难记住

  1. 不要理直气壮地聊敏感数字。
