еҸҜи§ӮжөӢжҖ§

зҺ°д»Јзҡ„еҸҜи§ӮжөӢжҖ§пјҲObservabilityпјүзӯүдәҺд»ҘдёӢдёүиҖ…зҡ„йӣҶеҗҲгҖӮ

  • еҸҜи§ӮжөӢжҖ§жҰӮеҝөиө·жәҗдәҺжҺ§еҲ¶зҗҶи®әдёӯзҡ„еҸҜи§ӮеҜҹжҖ§пјҲobservabilityпјү
    • зі»з»ҹзҡ„еҸҜи§ӮжөӢжҖ§пјҢжҢҮиҜҘзі»з»ҹеҸҜд»Ҙз”ұе…¶еӨ–йғЁиҫ“еҮәжҺЁж–ӯе…¶еҶ…йғЁзҠ¶жҖҒзҡ„зЁӢеәҰ
    • жҹҗдёҖдёӘзі»з»ҹзҡ„еҸҜи§ӮжөӢжҖ§и¶ҠејәпјҢйӮЈд№ҲжҲ‘д»¬еҜ№иҝҷдёӘзі»з»ҹзҡ„жҠҠжҺ§иғҪеҠӣд№ҹдјҡи¶Ҡејә

йңҖиҰҒжҳҺзЎ®пјҢзӣ‘жҺ§жҳҜдёҖдёӘеҠЁдҪңгҖӮ

  • зӣ®еүҚи®ӨдёәпјҢдёҖдёӘе®Ңж•ҙзҡ„еҸҜи§ӮжөӢжҖ§зі»з»ҹпјҢе…·еӨҮиҫ“еҮәдёүз§Қзұ»еһӢж•°жҚ®зҡ„иғҪеҠӣпјҡmetricsгҖҒlogгҖҒtraceгҖӮ
  • е®һйҷ…дёҠпјҢmetricsгҖҒlogгҖҒtraceеҸӘжҳҜдёүз§Қж•°жҚ®зұ»еһӢпјҢжң¬иә«дёҺеҸҜи§ӮжөӢжҖ§ж— е…ігҖӮд»…д»…жҳҜж”¶йӣҶиҝҷдәӣзұ»еһӢзҡ„ж•°жҚ®пјҢе№¶дёҚиғҪдҝқиҜҒзі»з»ҹзҡ„еҸҜи§ӮжөӢжҖ§гҖӮ
  • дҪҶжҳҜе°Ҷиҝҷдәӣж•°жҚ®иҝӣиЎҢеҗҲзҗҶзҡ„ж”¶йӣҶгҖҒеӯҳеӮЁгҖҒеҲҶжһҗе’ҢеҲ©з”ЁпјҢе°ұе®№жҳ“дҪҝзі»з»ҹе…·жңүдёҖе®ҡзҡ„еҸҜи§ӮжөӢжҖ§гҖӮеӣ жӯӨиҝҷдәӣж•°жҚ®пјҢеҸҲиў«з§°дёәеҸҜи§ӮжөӢж•°жҚ®гҖӮ

е®ҡд№үпјҡ

  • metricsжҢҮж ҮпјҡжҳҜеҺҹеӯҗжҖ§зҡ„гҖҒеҸҜзҙҜеҠ зҡ„дёҖз§ҚеҖјпјҢз”ЁдәҺиЎЁзӨәеңЁвјҖж®өж—¶й—ҙеҶ…жөӢйҮҸзҡ„ж•°еҖјпјҢжҜҸдёӘmetricsйғҪжҳҜдёҖдёӘйҖ»иҫ‘и®ЎйҮҸеҚ•е…ғгҖӮ
  • traceй“ҫи·ҜиҝҪиёӘпјҡеҸҲз§°еҲҶеёғејҸй“ҫи·ҜиҝҪиёӘпјҢиЎЁзӨә иҜ·жұӮйҖҡиҝҮеҲҶеёғејҸзі»з»ҹ зҡ„з«ҜеҲ°з«Ҝзҡ„и·Ҝеҫ„гҖӮ
  • logж—Ҙеҝ—пјҡз”ЁдәҺжҸҸиҝ°дёҖдәӣзҰ»ж•Јзҡ„пјҲдёҚиҝһз»ӯзҡ„пјүдәӢд»¶пјҢжҳҜеҜ№зү№е®ҡж—¶й—ҙеҸ‘вҪЈзҡ„дәӢд»¶зҡ„вҪӮжң¬и®°еҪ•гҖӮ

зү№жҖ§пјҡ

  • metrics еҖҫеҗ‘дәҺжӣҙиҠӮзңҒиө„жәҗпјҢеӣ дёәд»–дјҡвҖңеӨ©з„¶зҡ„вҖқеҺӢзј©ж•°жҚ®гҖӮ
  • logging еҖҫеҗ‘дәҺж— йҷҗеўһеҠ зҡ„пјҢдјҡйў‘з№Ғзҡ„и¶…еҮәйў„жңҹзҡ„е®№йҮҸгҖӮпјҲеҸҜд»ҘйҖҡиҝҮ е‘ҠиӯҰе’Ңе®ҡжңҹжё…зҗҶ иҝӣиЎҢйў„йҳІпјү
  • Tracing ж•°жҚ®еҲҷд»ӢдәҺдёӨиҖ…д№Ӣй—ҙ

Metric

Counter

  • CounterпјҢйҖӮз”ЁдәҺж ·жң¬ж•°жҚ®еҚ•и°ғйҖ’еўһзҡ„жҢҮж ҮпјҢеҚіеҸӘеўһдёҚеҮҸпјҢйҷӨйқһзӣ‘жҺ§зі»з»ҹеҸ‘з”ҹдәҶйҮҚзҪ®
  • з”ЁйҖ”пјҡжңҚеҠЎзҡ„иҜ·жұӮж•°гҖҒе·Іе®ҢжҲҗзҡ„д»»еҠЎж•°гҖҒй”ҷиҜҜеҸ‘з”ҹзҡ„ж¬Ўж•°зӯү

Gauge

  • GaugeпјҢйҖӮз”ЁдәҺж ·жң¬ж•°жҚ®еҸҜд»Ҙд»»ж„ҸеҸҳеҢ–зҡ„жҢҮж ҮпјҢеҚіеҸҜеўһеҸҜеҮҸ
  • з”ЁйҖ”пјҡжҜ”еҰӮ CPUжё©еәҰгҖҒеҶ…еӯҳдҪҝз”ЁзҺҮгҖҒеҪ“еүҚе№¶еҸ‘иҜ·жұӮзҡ„ж•°йҮҸзӯү

Timer

  • TimerпјҢйҖӮз”ЁдәҺйңҖиҰҒеҜ№жҹҗдёӘдәӢд»¶зҡ„еӨ„зҗҶж—¶й—ҙиҝӣиЎҢз»ҹи®Ўж—¶дҪҝз”Ё
  • з”ЁйҖ”пјҡжҜ”еҰӮ е…ій”®дёҡеҠЎзҡ„еӨ„зҗҶж—¶й—ҙгҖҒж•°жҚ®еә“и°ғз”ЁиҖ—ж—¶зӯү

TimerдёҖиҲ¬дјҡеҗҢж—¶и®°еҪ•CounterпјҢеңЁжңүдәӣз»„д»¶дёӯпјҢTimer == Histogram

Histogram

  • HistogramпјҢйҖӮз”ЁдәҺйңҖиҰҒеҜ№ж ·жң¬ж•°жҚ®иҝӣиЎҢдёҖе®ҡзҡ„з»ҹи®Ўе·ҘдҪңж—¶дҪҝз”Ё
    • Histogram еңЁдёҖж®өж—¶й—ҙиҢғеӣҙеҶ…еҜ№ж•°жҚ®иҝӣиЎҢйҮҮж ·пјҲйҖҡеёёжҳҜиҜ·жұӮжҢҒз»ӯж—¶й—ҙжҲ–е“Қеә”еӨ§е°Ҹзӯүпјү
    • е№¶е°Ҷе…¶и®Ўе…ҘеҸҜй…ҚзҪ®зҡ„еӯҳеӮЁжЎ¶пјҲbucketпјүдёӯпјҢеҗҺз»ӯеҸҜйҖҡиҝҮжҢҮе®ҡеҢәй—ҙзӯӣйҖүж ·жң¬
    • д№ҹеҸҜд»Ҙз»ҹи®Ўж ·жң¬жҖ»ж•°пјҢжңҖеҗҺдёҖиҲ¬е°Ҷж•°жҚ®еұ•зӨәдёәзӣҙж–№еӣҫ

Histogramи§ЈеҶій•ҝе°ҫй—®йўҳ

  • Histogram е’Ң Summary йғҪжҳҜдёәдәҶи§ЈеҶій•ҝе°ҫй—®йўҳиҖҢжҸҗеҮәзҡ„
  • еҫҲеӨҡжғ…еҶөдёӢпјҢжҲ‘д»¬еҖҫеҗ‘дәҺдҪҝз”Ёе№іеқҮеҖјеҺ»жҸҸиҝ°дёҖдәӣз»ҹи®Ўж•°жҚ®пјҢдҫӢеҰӮCPUзҡ„е№іеқҮдҪҝз”ЁзҺҮгҖҒйЎөйқўзҡ„е№іеқҮе“Қеә”ж—¶й—ҙ
    • з„¶иҖҢе№іеқҮеҖјжңүдёӘиҮҙе‘ҪзјәзӮ№пјҢйҡҫд»ҘеҜҹи§үе…¶дёӯзҡ„жһҒз«Ҝж•°жҚ®пјҢиҖҢдё”дјҡеӣ жһҒз«Ҝж•°жҚ®еҜјиҮҙе№іеқҮеҖјзҡ„еҮҶзЎ®жҖ§йҷҚдҪҺгҖӮ
    • жҜ”еҰӮпјҡеҰӮжһңеӨ§еӨҡж•°APIиҜ·жұӮйғҪз»ҙжҢҒеңЁ100msзҡ„е“Қеә”ж—¶й—ҙиҢғеӣҙеҶ…пјҢиҖҢдёӘеҲ«иҜ·жұӮзҡ„е“Қеә”ж—¶й—ҙйңҖиҰҒ5sпјҢжӯӨж—¶дҪҝз”Ёе№іеқҮеҖје°ұдјҡеҮәй—®йўҳгҖӮиҝҷе°ұжҳҜй•ҝе°ҫй—®йўҳгҖӮ
    • дҪҶжҳҜеҰӮжһңдҪҝз”ЁеҲҶдҪҚеҖјпјҢи®Ўз®—еҮә95%ж•°жҚ®зҡ„жңҖеӨ§еҖјпјҢеҸ‘зҺ°иҖ—ж—¶йғҪеңЁ100msд»ҘеҶ…пјҢиҜҙжҳҺ5sж•°жҚ®д»…д»…жҳҜдёӘеҲ«зҺ°еңәгҖӮ

Summary

  • дёҺ Histogram зұ»еһӢзұ»дјјпјҢSummary д№ҹжҳҜз”ЁдәҺиЎЁзӨәдёҖж®өж—¶й—ҙеҶ…зҡ„ж•°жҚ®йҮҮж ·з»“жһңпјҲйҖҡеёёжҳҜиҜ·жұӮжҢҒз»ӯж—¶й—ҙжҲ–е“Қеә”еӨ§е°ҸзӯүпјүгҖӮ
  • дҪҶдёҺ HistogramдёҚеҗҢзҡ„жҳҜпјҡSummary зӣҙжҺҘеӯҳеӮЁдәҶеҲҶдҪҚж•°пјҢдёҚйңҖиҰҒеҶҚйҖҡиҝҮеҢәй—ҙжқҘи®Ўз®—гҖӮ
  • Histogramе’ҢSummaryзҡ„ејӮеҗҢгҖӮ
    • зӣёеҗҢпјҡйғҪжңү_sumе’Ңcount
    • дёҚеҗҢпјҡйғҪжҸҗдҫӣдәҶеҲҶдҪҚж•°зҡ„и®Ўз®—пјҢдҪҶжҳҜж–№ејҸдёҚеҗҢ
      • Histogram жҳҜйҖҡиҝҮи®°еҪ•еҫҲеӨҡдёӘbucketзҡ„еҖјпјҢдҫӣжңҚеҠЎз«Ҝ йҖҡиҝҮеҢәй—ҙеҺ»и®Ўз®—еҲҶдҪҚж•°пјҢжҜҸдёӘжЎ¶еҶ…й»ҳи®ӨдёәзәҝжҖ§еҲҶеёғпјҢеӯҳеңЁдёҖе®ҡи®Ўз®—иҜҜе·®гҖӮ
      • Summary жҳҜеңЁе®ўжҲ·з«ҜзӣҙжҺҘеҫ—еҲ°дәҶеҲҶдҪҚж•°пјҢжңҚеҠЎз«Ҝж— йңҖи®Ўз®—пјҢеҲҶдҪҚж•°жҳҜзӣҙжҺҘж №жҚ®ж•°жҚ®жҺ’еәҸеҫ—еҲ°зҡ„пјҢжҳҜеҮҶзЎ®зҡ„гҖӮ

OpenTelemetry

OpenTelemetry зҡ„з»ҲжһҒзӣ®ж ҮжҳҜжҠҠдәәд»¬д№ӢеүҚжүӢе·ҘеҒҡзҡ„пјҢйҖҡиҝҮжһ¶жһ„и®ҫи®Ўжҗӯе»әдёҚеҗҢзҡ„жҠҖжңҜз»„д»¶жқҘи§ӮжөӢзі»з»ҹзҠ¶жҖҒзҡ„ж–№жЎҲз»ҹеҗҲжҲҗдёҖдёӘеӨ§дёҖз»ҹзҡ„ж–№жЎҲпјҢжҳҜе®һи·өйўҶе…ҲдәҺзҗҶи®әпјҢзҗҶи®әеҸҲз»ҹдёҖе®һи·өзҡ„дёҖдёӘеҘҪдҫӢеӯҗгҖӮ

OpenTelemetryзҡ„иҮӘиә«е®ҡдҪҚеҫҲжҳҺзЎ®пјҡж•°жҚ®йҮҮйӣҶе’Ңж ҮеҮҶи§„иҢғзҡ„з»ҹдёҖпјҢеҜ№дәҺж•°жҚ®еҰӮдҪ•еҺ»дҪҝз”ЁгҖҒеӯҳеӮЁгҖҒеұ•зӨәгҖҒе‘ҠиӯҰпјҢе®ҳж–№жҳҜдёҚж¶үеҸҠзҡ„пјҢжҲ‘д»¬зӣ®еүҚжҺЁиҚҗдҪҝз”ЁPrometheus + GrafanaеҒҡMetricsеӯҳеӮЁгҖҒеұ•зӨәпјҢдҪҝз”ЁJaegerеҒҡеҲҶеёғејҸи·ҹиёӘзҡ„еӯҳеӮЁе’Ңеұ•зӨәгҖӮ

APMзҡ„дёӨз§Қзӣ‘жҺ§еӯҗзұ»пјҡеҲҶеёғејҸи·ҹиёӘе’ҢMetricsпјҢе…¶е®һиҝҳжңүз¬¬дёүз§ҚпјҢе°ұжҳҜLoggingж—Ҙеҝ—пјҢзӣ®еүҚеёёи§Ғзҡ„ж—Ҙеҝ—ж”¶йӣҶе№іеҸ°жңүEFKпјҲжҠҠ ELK йҮҢзҡ„ Logstash жҚўжҲҗдәҶ filebeatпјүгҖҒFluentdгҖӮ

OpenTelemetry жңүдёҖдёӘеҲӣж–°пјҢMTL дёүйЎ№ж•°жҚ®иў«з»ҹдёҖеңЁ baggage йҮҢйқўдәҶгҖӮ
еҸӮиҖғпјҡгҖҠOpenTelemetry-еҸҜи§ӮеҜҹжҖ§зҡ„ж–°ж—¶д»ЈгҖӢ

OpenTelemetry дё»иҰҒи§ЈеҶідә‘еҺҹз”ҹеҸҜи§ӮжөӢжҖ§пјҲзӣ‘жҺ§пјҢеҲҶеёғејҸиҝҪиёӘпјҢж—Ҙеҝ—пјүзҡ„ж ҮеҮҶеҲ¶е®ҡе’Ңе·Ҙе…·е»әи®ҫдёӨеӨ§зұ»й—®йўҳгҖӮ

й—®йўҳжӢҶи§Јпјҡ

  1. з»ҹдёҖеҸҜи§ӮжөӢжҖ§йўҶеҹҹж ҮеҮҶпјҡеҚҸи®®пјҲAPIпјүпјҢдј иҫ“ж•°жҚ®жЁЎеһӢгҖӮ
  2. дёәдёҚеҗҢжҠҖжңҜжҸҗдҫӣз»ҹдёҖзҡ„ SDKпјҢз»ҹдёҖз”ЁжҲ·жҺҘе…ҘдҪ“йӘҢгҖӮ
  3. йҷҚжң¬еўһж•ҲпјҢеҸҜи§ӮжөӢжҖ§е№іеҸ°жҠҖжңҜе…ұдә«пјҢеҠҹиғҪз»„д»¶еӨҚз”ЁпјҢиғҪеҠӣеҚҸеҗҢгҖӮ
  4. дёҡеҠЎиҗҪең°пјҢжҺЁеҠЁеҸҜи§ӮжөӢжҖ§йўҶеҹҹе…ҲиҝӣжҠҖжңҜиғҪеҠӣиҗҪең°дёҡеҠЎгҖӮ

з»ҹдёҖж•°жҚ®жЁЎеһӢе’Ңж•°жҚ®дј иҫ“ж ҮеҮҶпјҡ

  • ж—Ҙеҝ—жЁЎеқ— log
  • иҝҪиёӘжЁЎеқ— tracing
  • зӣ‘жҺ§жЁЎеқ— monitor/metrics
  • жҖ§иғҪжЁЎеқ— APM performance

Automatic Instrumentation

жҢүз…§и®ҫи®ЎпјҢOpenTelemetry зҡ„ agent еңЁеҗҜеҠЁдёӯеҠ е…ҘеҗҺе°ұдјҡиҮӘеҠЁең°еҒҡжӢҰжҲӘпјҡjava -javaagent:path/to/opentelemetry-javaagent.jar \ -jar myapp.jarгҖӮ

е‘Ҫд»ӨиЎҢйҖүйЎ№й…ҚзҪ®

иҝҷдёӘ agent зҡ„иЎҢдёәжҳҜеҸҜд»ҘйҖҡиҝҮйҖүйЎ№пјҲoptionпјүй«ҳеәҰеҸҜй…ҚзҪ®зҡ„пјҲhighly configurableпјүпјҡ

еҸҜд»ҘзңӢеҲ° zipkin жҳҜиў«еҪ“еҒҡдёҖдёӘеҸҜйҖүзҡ„ exporter и®ҫзҪ®иө·жқҘзҡ„пјҢжҲ‘д»¬д№ҹеҸҜд»Ҙй…ҚзҪ®жҲ‘д»¬зҡ„ service.nameгҖӮ

зҺҜеўғеҸҳйҮҸй…ҚзҪ®

еҪ“з„¶пјҢдҪҝз”ЁзҺҜеўғеҸҳйҮҸд№ҹжҳҜеҸҜд»Ҙзҡ„пјҡ

д№ҹеҸҜд»ҘдҪҝз”Ёй…ҚзҪ®ж–Үд»¶пјҲиҝҷзӮ№е°ұеҫҲеғҸ ES дәҶпјүпјҡ

@WithSpan

иҝҷдёӘжіЁи§ЈеҜ№еҗҢжӯҘж–№жі•и°ғз”ЁдјҡиҮӘеҠЁжӢјжҺҘеҮә<className>.<methodName>пјҢиҖҢдё”иғҪеӨҹйҖӮеә”future- or promise-likeзҡ„иҜ·жұӮпјҲthe span will not be ended until the future completes.пјүгҖӮ

@SpanAttribute дјҡжҠҠж–№жі•еҸӮж•°иҮӘеҠЁеҠ иҝӣ span зҡ„ AttributeMap йҮҢйқўпјҡ

иҝҷдёӘжіЁи§ЈеҸҜд»Ҙиў« suprressedпјҡOTEL_INSTRUMENTATION_OPENTELEMETRY_ANNOTATIONS_EXCLUDE_METHODS=my.package.MyClass1[method1,method2];my.package.MyClass2[method3]гҖӮ

ж— дҫөе…Ҙең°йҖҡиҝҮ include жҠҠж–№жі•жӢҰжҲӘиө·жқҘпјҡOTEL_INSTRUMENTATION_METHODS_INCLUDE=my.package.MyClass1[method1,method2];my.package.MyClass2[method3]гҖӮ

й…ҚзҪ®йЎ№

Agent ConfigurationйҮҢзҡ„й…ҚзҪ®жҳҜжһҒжҳ“еҸҳеҠЁзҡ„пјҢжңҖеҘҪе®һж—¶жҹҘиҜўдёәеҘҪгҖӮзҺҜеўғеҸҳйҮҸзӯүдәҺеӨ§еҶҷеӯ—жҜҚеҠ дёӢеҲ’зәҝпјҲOTEL_TRACES_EXPORTERпјүпјҢиҖҢй…ҚзҪ®ж–Үд»¶жҳҜжҢүз…§ properties ж јејҸпјҲе°ҸеҶҷеӯ—жҜҚеҠ .пјүжқҘи®ҫи®Ўзҡ„пјҲotel.traces.exporterпјүгҖӮ

зҰҒз”Ёжү“зӮ№

жңүдёҖдәӣжү“зӮ№й»ҳи®ӨжҳҜ disabled зҡ„пјҢиҝҷзӮ№е’Ң raptor д№ҹжңүзӮ№еғҸгҖӮдёәд»Җд№ҲдёҚйҮҮз”Ёи°ғж•ҙйҮҮж ·зҺҮжҲ–иҖ…еҺӢзј©пјҲжҜ”еҰӮеҺӢзј©дёәз®ҖеҚ•зҡ„ countпјүпјҢжҲ–иҖ…йҷҗеҲ¶ tagValue зҡ„жЁЎејҸжқҘи§ЈеҶіиҝҷдёӘй—®йўҳпјҹ

зү№еҲ«йҮҚиҰҒзҡ„ Manual Instrumentation

еҰӮжһңжңүдёҖдёӘеә“пјҲLibrariesпјүжғіиҰҒдҪҝз”Ё OpenTelemetry жқҘеҜјеҮәйҒҘжөӢж•°жҚ®пјҢе®ғеҝ…йЎ»еҸӘдҫқиө–opentelemetry-apiеҢ…пјҢз»қдёҚиҰҒдҫқиө–дәҺ OpenTelemetry SDKгҖӮOpenTelemetry SDK еҝ…йЎ»з”ұеә”з”Ё**зЁӢеәҸпјҲApplicationsпјү**йҖҡиҝҮдҫқиө–дәҺ OpenTelemetry SDK жқҘжҸҗдҫӣгҖӮиҝҷж ·пјҢreal implementation еҸӘжңү the user application is configured for it жүҚдјҡиў«з”ЁеҲ°пјҢиҝҷе°ұжҸҗдҫӣдәҶ api е’Ң implementation зҡ„еҲҶзҰ»гҖӮ

е®ҳж–№ж–ҮжЎЈдё»иҰҒй’ҲеҜ№opentelemetry-sdkзҡ„е®һзҺ°зҡ„й…ҚзҪ®жі•пјҢе…¶д»–е®һзҺ°зҡ„й…ҚзҪ®еҸҜд»ҘйҮҮз”ЁдёҚеҗҢзҡ„ж–№жі•гҖӮ

дёҖдәӣеҖјеҫ—дёҖжҸҗзҡ„и®ҫи®ЎеҮҶеҲҷпјҲDesign Princinpalsпјү
end-users жіЁе…Ҙе®һдҫӢ

OpenTelemetry йј“еҠұзұ»еә“зҡ„и®ҫи®ЎиҖ…е…Ғи®ё end-users inject дёҖдёӘ OpenTelemetry зҡ„е®һдҫӢеҲ° instrumentation code йҮҢпјҲиҝҷзұ»дјј yield е’Ң SPIпјүпјҢиҝҷз§Қи®ҫи®ЎеҸҜд»ҘеӣһйҖҖеҲ°д»Һ GlobalOpenTelemetry иҺ·еҸ–дёҖдёӘ instanceгҖӮend-users дёҚиғҪй…ҚзҪ®иҝҷдёӘ instanceпјҢжүҖд»ҘиҝҷжҳҜдёҖдёӘеҫҲи„ҶејұпјҲbrittleпјүзҡ„йҖүжӢ©гҖӮ

дёҚиҰҒдҪҝз”Ё OpenTelemetry SDK йҮҢзҡ„ж–№жі•

жңҖеҘҪдҪҝз”Ё OpenTelemetry API йҮҢзҡ„ж–№жі•гҖӮ
